# P0102 Constant Palindrome Sum 题解

## 一、题意简化

本题的核心是：给定偶数长度数组 a，所有元素都在 1 到 k 之间。

## 二、算法设计

每对对称元素 (u, v) 对目标和 x 的贡献分三类：

- 若 x = u+v，修改 0 次；
- 若 min(u,v)+1 <= x <= max(u,v)+k，修改 1 次；
- 其他情况修改 2 次。

先假设每对都修改 2 次，再用差分数组对“一次修改区间”减 1，对“零次修改点”再减 1。枚举 x 取最小值。

## 四、复杂度分析

每组 O(n+k)。
