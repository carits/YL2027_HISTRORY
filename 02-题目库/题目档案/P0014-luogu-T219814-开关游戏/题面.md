# P0014 开关游戏

## 题目描述

有$n$盏灯依次编号为$1$到$n$，初始时都是关闭的状态。

接下来玩一个游戏，对灯进行$k$轮操作，第$i$轮操作按下所有编号为$i$的倍数的灯的开关。

比如，第$1$轮按下所有灯的开关，第$2$轮按下所有编号为$2$的倍数的灯的开关……

每次按下开关后灯的状态就会变化一次。

求最后有哪些灯是开着的，将编号由小到大进行输出。

## 输入格式

输入只有一行，包含两个正整数$n$、$k$，之间用空格隔开。

## 输出格式

输出只有一行，包含若干个正整数，之间用空格隔开。

## 说明与数据范围

对于100%的数据，$1 ≤ n ≤ 10000$。

## 样例

### 样例 1

**输入**

```text
4 2
```text

**输出**
```text
1 3
```
