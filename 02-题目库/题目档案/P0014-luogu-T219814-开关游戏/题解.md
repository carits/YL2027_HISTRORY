# P0014 开关游戏 题解

## 一、题意简化

本题可以归入 **模拟**。核心任务是：根据输入限制，选择合适的数据结构或算法，避免直接写出超出复杂度的暴力做法。

## 二、思路分析

直接维护每盏灯的开关状态。第 $i$ 轮枚举 $i,2i,3i,\ldots$ 这些编号并翻转即可。

## 三、算法步骤

1. 读入题目数据。
2. 根据题意维护必要的状态或预处理信息。
3. 按照上述思路完成计算。
4. 输出题目要求的答案，注意空格、换行和特殊输出格式。

## 四、正确性说明

算法直接围绕题目中的不变量或等价转化进行：每一步维护的状态都与原问题中的真实状态一一对应；当状态更新结束时，得到的就是题目要求的最终结果。对于排序、枚举、模拟、动态规划或数学转化类题目，代码中的比较规则、转移规则或公式均来自题意本身，因此不会改变答案含义。

## 五、复杂度分析

$O(n\log n)$ 量级，满足 $n\le 10000$。

## 六、参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, k;
    cin >> n >> k;
    vector<int> on(n + 1, 0);
    for (int i = 1; i <= k; i++) {
        for (int j = i; j <= n; j += i) on[j] ^= 1;
    }
    bool first = true;
    for (int i = 1; i <= n; i++) {
        if (on[i]) {
            if (!first) cout << ' ';
            first = false;
            cout << i;
        }
    }
    cout << '\n';
    return 0;
}
```
