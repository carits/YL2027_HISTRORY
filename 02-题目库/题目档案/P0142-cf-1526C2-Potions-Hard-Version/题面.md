# P0142 Potions (Hard Version) 题面

- 平台：CodeForces
- 题号：1526C2

## 问题陈述

给定 n 瓶药水，每瓶药水有一个数值，可以为正也可以为负。你从左到右考虑药水，可以选择喝或不喝。

要求任意时刻当前生命值不能为负，初始生命值为 0。求最多能喝多少瓶。

## 输入格式

第一行输入 n。

第二行输入 n 个整数。

## 输出格式

输出最多能喝的药水数量。

## 数据范围

1 <= n，药水值可正可负。

## 样例输入

```text
6
4 -4 1 -3 1 -3
```

## 样例输出

```text
5
```
