# P0132 Points and Minimum Distance 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 贪心 |
| 子标签 | 排序, 构造坐标, 曼哈顿距离 |
| 三维难度 | 2/2/2 |
| 综合等级 | L2 |
| 推荐阶段 | 普及- |
| 适合用途 | 排序构造题 |

## 训练价值

这题适合讲曼哈顿距离按维度拆开，以及“构造达到下界”的证明。

## 讲评重点

1. 先让学生复述题意和限制条件。
2. 明确核心状态、枚举对象或判定函数。
3. 单独讲边界和常见错误。
4. 最后再给代码，避免学生只背实现。
