## 问题陈述

一把铲子价格为 k。商店只收整十元或额外一枚面值 r 的硬币。求最少买多少把铲子，使得总价的个位为 0 或 r。

## 输入格式

输入 k, r。

## 输出格式

输出最少购买数量。

## 自测样例

### 输入

```text
117 3
```

### 输出

```text
9
```
