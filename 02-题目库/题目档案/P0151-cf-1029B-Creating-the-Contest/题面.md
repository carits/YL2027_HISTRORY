# P0151 Creating the Contest 题面

## 题目来源

- 平台：CodeForces
- 题号：1029B
- 原题链接：https://codeforces.com/problemset/problem/1029/B
- VJudge 写法：`[problem:CodeForces-1029B]`

## 问题陈述

给定一个已按难度非降序排列的题目序列。一个连续题组是合法的，当其中相邻两题都满足后一道难度不超过前一道难度的两倍。求最长合法连续题组长度。

## 输入格式

第一行输入 n。第二行输入 n 个整数。

## 输出格式

输出最长长度。

## 约束要点

线性扫描即可。

## 样例输入

```text
5
1 2 3 8 9
```

## 样例输出

```text
3
```
