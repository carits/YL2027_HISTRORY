# P0114 Candies 题面

## 题目来源

- 平台：CodeForces
- 题号：1343A
- 原题链接：https://codeforces.com/problemset/problem/1343/A
- VJudge 写法：`[problem:CodeForces-1343A]`

## 问题陈述

给定 n。已知 n 可以表示为 x * (2^k - 1)，其中 k > 1。

求任意一个满足条件的 x。

## 输入格式

第一行输入 T。

接下来 T 行，每行一个 n。

## 输出格式

每组输出一个满足条件的 x。

## 数据范围

n 为正整数，并保证存在答案。

## 样例输入

```text
3
3
7
21
```

## 样例输出

```text
1
1
7
```
