# P0070 Blackslex and Password 题解

## 一、题意简化

密码只能使用前 k 个小写字母，并要求任意两个位置差为 x 的倍数时字符不同。求最小的长度 n，使得不存在合法密码。

## 二、算法设计

位置按照下标模 x 分成 x 类。同一类中任意两位置差都是 x 的倍数，因此同一类最多放 k 个字符。总长度最多为 kx，所以最小不可行长度为 kx+1。

## 三、复杂度分析

O(1)。

## 四、常见错误

- 多组数据没有清空数组、计数器或集合。
- 输出大小写、空格、换行不符合题目要求。
- 没有按数据范围选择合适的数据类型。
- 忽略极端数据，例如空结果、只有一个元素、全部相同、边界上限等。
