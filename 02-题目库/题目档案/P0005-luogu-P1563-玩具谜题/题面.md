# P0005 玩具谜题 题面

## 问题陈述

有 n 个玩具小人按逆时针顺序围成一圈。每个小人有朝向：0 表示朝向圈内，1 表示朝向圈外，并且有一个职业名。从第 1 个小人开始，依次执行 m 条指令。每条指令给出 $a$ 和 $s$：a=0 表示从当前小人的左边数 s 个，a=1 表示从当前小人的右边数 s 个。由于朝内和朝外时左右方向相反，需要根据当前小人的朝向决定位置增减。求执行完所有指令后所在小人的职业名。

## 输入格式

第一行输入两个整数 n, m。
接下来 $n$ 行每行输入一个整数 d_i 和一个字符串 name_i，表示第 $i$ 个小人的朝向和职业名。
接下来 $m$ 行每行输入两个整数 a_i, s_i，表示一条指令。

## 输出格式

输出最后到达的小人的职业名。

## 自测样例

### 输入

```text
7 3
0 singer
0 reader
0 mengbier
1 thinker
1 archer
0 writer
1 mogician
0 3
1 1
0 2
```

### 输出

```text
writer
```
