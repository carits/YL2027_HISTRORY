# P0196 Anti-Fibonacci Permutation 题面

## 问题陈述

对每个 `n`，输出 `n` 个 `1..n` 的排列，使得每个排列都不是斐波那契式排列，即不存在连续三个位置满足前两项之和等于第三项。

## 输入格式

第一行输入测试组数。每组输入 `n`。

## 输出格式

每组输出 `n` 行排列。

## 数据范围

`n` 为正整数，所有输出满足题目格式即可。

## 样例输入

```text
1
3
```

## 样例输出

```text
3 2 1
2 3 1
2 1 3
```
