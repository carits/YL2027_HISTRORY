# P0196 Anti-Fibonacci Permutation 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 构造 |
| 子标签 | 排列构造, 相邻交换, 反例规避 |
| 三维难度 | 1/2/1 |
| 综合等级 | L2 |
| 推荐阶段 | 入门-普及 |
| 适合用途 | 构造入门题 |

## 训练价值

适合讲构造题里“先找一个稳定形态，再做可控扰动”。

## 课堂讲解建议

1. 先让学生复述题意中的限制条件。
2. 再引导学生抽象出核心状态或判定函数。
3. 对照样例手推一遍，确认边界情况。
4. 最后再讲代码结构和复杂度。

## 二刷建议

完成本题后，可以要求学生不看题解重新写一遍，并单独解释为什么算法不会漏情况。
