# P0018 最大子段和

## 题目描述

给定$n$个数组成的整数数列$a_i$，求所有的子段中的最大总和。

子段指位置连续的一段数，可以为空。

## 输入格式

第一行包含整数$n$。

第二行包含$n$个整数$a_i$。

## 输出格式

只有一行，包含最大子段和。

## 说明与数据范围

对于50%的数据，$1 ≤ n ≤ 500$。

对于100%的数据，$1 ≤ n ≤ 5000$，$-10000 ≤ a_i ≤ 10000$。

## 样例

### 样例 1

**输入**

```text
5 
2 -1 2 -3 2
```text

**输出**
```text
3
```
