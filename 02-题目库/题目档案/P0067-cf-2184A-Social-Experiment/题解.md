# P0067 Social Experiment 题解

## 一、题意简化

本题可以简化为：

```text
有 n 个人，需要分成若干个 2 人或 3 人的小队。每个小队整体选择两个阵营之一。求两个阵营人数差的最小可能值。
```

## 二、算法设计

除 n=2 和 n=3 外，可以把两边人数尽量平分；偶数能做到 0，奇数能做到 1。n=2 只能形成一个 2 人队，答案为 2；n=3 只能形成一个 3 人队，答案为 3。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `数学` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

每组 O(1)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
