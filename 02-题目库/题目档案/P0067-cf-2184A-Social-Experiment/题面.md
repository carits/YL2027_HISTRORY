# P0067 Social Experiment 题面

## 来源

- 平台：CodeForces
- 题号：2184A
- 原题链接：https://codeforces.com/problemset/problem/2184/A
- VJudge 写法：`[problem:CodeForces-2184A]`

## 问题陈述

有 n 个人，需要分成若干个 2 人或 3 人的小队。每个小队整体选择两个阵营之一。求两个阵营人数差的最小可能值。

## 输入格式

第一行输入测试组数 T。每组输入一个整数 n。

## 输出格式

每组输出最小人数差。

## 样例输入

```text
3
2
5
12
```

## 样例输出

```text
2
1
0
```
