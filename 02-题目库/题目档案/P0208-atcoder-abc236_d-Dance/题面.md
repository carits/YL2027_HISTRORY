# P0208 Dance 题面

## 题目来源

- 平台：AtCoder
- 题号：abc236_d
- 原题链接：https://atcoder.jp/contests/abc236/tasks/abc236_d
- VJudge 写法：`[problem:AtCoder-abc236_d]`

## 问题描述

有 `2N` 个人，要分成 `N` 对。每对 `(i,j)` 有相性值 `A_{i,j}`，整个舞会乐趣是所有配对相性值的按位异或。求最大可能乐趣。

## 输入格式

输入 `N`，随后按上三角形式输入所有 `A_{i,j}`。

## 输出格式

输出最大异或值。

## 数据范围

`1 <= N <= 8`，`0 <= A_{i,j} < 2^30`。

## 样例输入

```text
2
4 0 1
5 3
2
```

## 样例输出

```text
6
```
