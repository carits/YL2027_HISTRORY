# P0179 Sushi for Two 题面

## 问题陈述

给定长度为 `n` 的序列，每个位置的寿司类型为 1 或 2。需要选择一个连续子段，使得子段前一半全是同一种类型，后一半全是另一种类型，并且两半长度相等。求最长合法子段长度。

## 输入格式

第一行输入 `n`。第二行输入 `n` 个整数，每个整数为 1 或 2。

## 输出格式

输出最长合法连续子段长度。

## 数据范围

`2 <= n <= 100000`。

## 样例输入

```text
6
2 2 1 1 2 2
```

## 样例输出

```text
4
```
