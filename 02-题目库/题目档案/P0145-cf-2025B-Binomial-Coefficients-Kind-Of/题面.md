# P0145 Binomial Coefficients, Kind Of 题面

## 题目来源

- 平台：CodeForces
- 题号：2025B
- 原题链接：https://codeforces.com/problemset/problem/2025/B
- VJudge 写法：`[problem:CodeForces-2025B]`

## 问题描述

错误的二项式递推为 `C[n][k] = C[n][k-1] + C[n-1][k-1]`，且边界 `C[n][0]=C[n][n]=1`。给出多组 `(n_i,k_i)`，求这个错误递推下的 `C[n_i][k_i] mod 1e9+7`。

## 输入格式

第一行输入 $t$。第二行输入 $t$ 个 $n_i$，第三行输入 $t$ 个 $k_i$。

## 输出格式

输出 $t$ 行，每行一个答案。

## 数据范围

`1 <= t <= 100000`，`2 <= n_i <= 100000`，`1 <= k_i < n_i`。

## 样例输入

```text
3
2 5 5
1 2 3
```

## 样例输出

```text
2
4
8
```
