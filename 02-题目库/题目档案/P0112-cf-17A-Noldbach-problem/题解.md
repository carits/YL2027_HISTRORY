# P0112 Noldbach problem 题解

## 一、题意简化

本题需要围绕 **数学**，处理 `质数筛, 相邻质数` 相关问题。

## 二、算法设计

先用筛法求出 n 以内质数。

枚举相邻质数 p_$i$ 和 $p$_{i+1}，计算 x = p_i + p_{i+1} + 1。如果 x <= n 且 x 本身是质数，就计数。最后比较计数和 k。

## 三、正确性说明

算法的核心是抓住题目中的不变量或最优选择依据，并在每一步只保留对后续有意义的信息。对于需要枚举的题目，枚举对象被限制在可证明的有限范围内；对于贪心题，选择依据保证不会让后续状态变差；对于动态规划题，状态转移完整覆盖所有可行路径。

## 四、复杂度分析

筛法 O(n log log n)，枚举 O(质数个数)。

## 五、常见错误

- 没有单独处理边界输入。
- 使用 `int` 导致中间结果溢出。
- 枚举范围过大或过小。
- 多组数据时没有清空临时变量。
