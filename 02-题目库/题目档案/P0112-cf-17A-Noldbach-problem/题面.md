# P0112 Noldbach problem 题面

## 题目来源

- 平台：CodeForces
- 题号：17A
- 原题链接：https://codeforces.com/problemset/problem/17/A
- VJudge 写法：`[problem:CodeForces-17A]`

## 问题陈述

给定 $n$ 和 k。

定义一种特殊质数：如果某个质数可以表示成两个相邻质数之和再加 1，即 p_i + p_{i+1} + 1，并且它不超过 n，那么它计入统计。

判断这样的质数个数是否至少为 k。

## 输入格式

输入两个整数 $n$ 和 $k$。

## 输出格式

如果数量至少为 k，输出 YES，否则输出 NO。

## 数据范围

1 <= n <= 1000。

## 样例输入

```text
27 2
```

## 样例输出

```text
YES
```
