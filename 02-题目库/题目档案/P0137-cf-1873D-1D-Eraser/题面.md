# P0137 1D Eraser 题面

## 题目来源

- 平台：CodeForces
- 题号：1873D
- 原题链接：https://codeforces.com/problemset/problem/1873/D
- VJudge 写法：`[problem:CodeForces-1873D]`

## 问题陈述

给定长度为 n 的字符串，只包含 $W$ 和 $B$。一次操作可以选择一个长度为 k 的连续区间，把其中所有字符变成 W。

求把所有 B 都变成 W 的最少操作次数。

## 输入格式

第一行输入 T。

每组输入 n、k 和字符串 s。

## 输出格式

每组输出最少操作次数。

## 数据范围

1 <= k <= n。

## 样例输入

```text
3
6 2
WBBWBW
5 3
BBBBB
4 1
WWWW
```

## 样例输出

```text
2
2
0
```
