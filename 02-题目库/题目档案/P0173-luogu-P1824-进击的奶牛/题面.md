# P0173 进击的奶牛 题面

## 题目来源

- 平台：洛谷
- 题号：P1824
- 原题链接：https://www.luogu.com.cn/problem/P1824
- VJudge 写法：`[problem:洛谷-P1824]`

## 问题陈述

给定 n 个牛棚的位置，要放入 c 头牛。要求任意两头牛之间的最小距离尽可能大，输出这个最大最小距离。

## 输入格式

第一行输入 n 和 c。接下来 n 行每行一个牛棚位置。

## 输出格式

输出最大最小距离。

## 约束要点

位置排序后进行二分答案。

## 样例输入

```text
5 3
1
2
8
4
9
```

## 样例输出

```text
3
```
