# P0189 平面最近点对（加强版） 题面

## 来源

- 平台：洛谷
- 题号：P1429
- 原题链接：https://www.luogu.com.cn/problem/P1429
- VJudge 写法：`[problem:洛谷-P1429]`

## 问题陈述

给定平面上的 $n$ 个点，求所有点对中距离最短的一对的距离。

## 输入格式

第一行输入 n。接下来 n 行每行输入点的坐标 x,y。

## 输出格式

输出最短距离，保留 4 位小数。

## 样例输入

```text
3
1 1
1 2
2 2
```

## 样例输出

```text
1.0000
```
