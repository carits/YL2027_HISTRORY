# P0189 平面最近点对（加强版） 题解

## 一、题意简化

本题可以简化为：

```text
给定平面上的 n 个点，求所有点对中距离最短的一对的距离。
```

## 二、算法设计

按 x 坐标排序并扫描，维护横坐标距离仍可能成为答案的候选点集合。集合按 y 坐标排序，只检查 y 差不超过当前最优距离的点。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `分治` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

O(n log n)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
