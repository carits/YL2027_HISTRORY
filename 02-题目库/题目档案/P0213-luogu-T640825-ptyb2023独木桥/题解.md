# P0213 【ptyb2023】独木桥 题解

## 一、题意简化

本题可以归入 **模拟**。核心任务是：根据输入限制，选择合适的数据结构或算法，避免直接写出超出复杂度的暴力做法。

## 二、思路分析

按时间模拟信号灯周期。每次根据当前时间判断哪一侧绿灯以及本段绿灯剩余时间；若当前侧无人或队首高素质村民无法在绿灯内通过，就跳到本段绿灯结束；否则让队首过桥，低素质村民只要绿灯未结束就会出发。

## 五、复杂度分析

$O(n+m)$。

## 六、参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    ll x, y;
    cin >> n >> m >> x >> y;
    queue<pair<ll, int>> q[2];
    for (int i = 0; i < n; i++) {
        ll t;
        int h;
        cin >> t >> h;
        q[0].push({t, h});
    }
    for (int i = 0; i < m; i++) {
        ll t;
        int h;
        cin >> t >> h;
        q[1].push({t, h});
    }

    ll now = 0, cycle = x + y;
    while (!q[0].empty() || !q[1].empty()) {
        ll r = now % cycle;
        int side;
        ll end;
        if (r < x) {
            side = 0;
            end = now + (x - r);
        } else {
            side = 1;
            end = now + (cycle - r);
        }
        if (q[side].empty()) {
            now = end;
            continue;
        }
        auto [t, h] = q[side].front();
        ll rem = end - now;
        if (h == 1 || t <= rem) {
            q[side].pop();
            now += t;
        } else {
            now = end;
        }
    }
    cout << now << '\n';
    return 0;
}
```
