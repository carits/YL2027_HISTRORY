# P0172 奶牛晒衣服 题面

## 题目来源

- 平台：洛谷
- 题号：P1843
- 原题链接：https://www.luogu.com.cn/problem/P1843
- VJudge 写法：`[problem:洛谷-P1843]`

## 问题描述

有 `n` 件湿衣服。每秒所有衣服都会自然减少 `a` 点湿度；同时只有一台烘干机，每秒可选择一件衣服额外减少 `b` 点湿度。求让所有衣服湿度降到 `0` 所需的最少时间。

## 输入格式

第一行输入 `n,a,b`。接下来 `n` 行，每行输入一件衣服初始湿度 `w_i`。

## 输出格式

输出最少时间。

## 数据范围

`1 <= n,a,b,w_i <= 5 * 10^5`。

## 样例输入

```text
3 1 2
1
2
5
```

## 样例输出

```text
2
```
