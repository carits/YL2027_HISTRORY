# P0006 [USACO1.1] 你的飞碟在这儿 题面

## 问题陈述

给出两个只含大写英文字母的字符串。把每个字母转成 A=1, B=2, ..., Z=26，然后分别计算两个字符串所有字母数值的乘积对 47 取模。若两个结果相同，输出 GO，否则输出 STAY。

## 输入格式

输入两行，每行一个大写字母字符串。

## 输出格式

若两个字符串的乘积模 47 相同，输出 GO；否则输出 STAY。

## 自测样例

### 输入

```text
COMETQ
HVNGAT
```

### 输出

```text
GO
```
