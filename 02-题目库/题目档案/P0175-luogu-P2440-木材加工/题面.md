# P0175 木材加工 题面

## 问题陈述

给定若干根原木长度，需要切出至少 `k` 段长度相同的小木段。木段长度必须是正整数，原木可以有剩余。求可以切出的最大小木段长度。如果长度为 1 的小段都无法切出，输出 0。

## 输入格式

第一行输入两个整数 `n, k`。接下来 `n` 行，每行一个整数 `L_i`，表示一根原木长度。

## 输出格式

输出一个整数，表示满足要求的最大小段长度。

## 数据范围

`1 <= n <= 100000`，`1 <= k <= 100000000`，`1 <= L_i <= 100000000`。

## 样例输入

```text
2 6
11
21
```

## 样例输出

```text
5
```
