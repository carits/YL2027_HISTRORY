# P0087 Portal 题解

## 一、题意简化

本题可以归入 **贪心**。核心任务是：根据输入限制，选择合适的数据结构或算法，避免直接写出超出复杂度的暴力做法。

## 二、思路分析

两个门户之间的元素只能整体循环位移，门户外的元素相对顺序不变。先把门户中间段旋转到字典序最小，再把这一段插入到外部序列中第一个大于段首的位置前面。

## 三、算法步骤

2. 根据题意维护必要的状态或预处理信息。
3. 按照上述思路完成计算。
4. 输出题目要求的答案，注意空格、换行和特殊输出格式。

## 四、正确性说明

算法直接围绕题目中的不变量或等价转化进行：每一步维护的状态都与原问题中的真实状态一一对应；当状态更新结束时，得到的就是题目要求的最终结果。对于排序、枚举、模拟、动态规划或数学转化类题目，代码中的比较规则、转移规则或公式均来自题意本身，因此不会改变答案含义。

## 五、复杂度分析

$O(n)$ 每组。

## 六、参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T;
    cin >> T;
    while (T--) {
        int n, x, y;
        cin >> n >> x >> y;
        x--;
        y--;
        vector<int> a, b;
        for (int i = 0; i < n; i++) {
            int v;
            cin >> v;
            if (i <= x || i > y) a.push_back(v);
            else b.push_back(v);
        }
        if (!b.empty()) rotate(b.begin(), min_element(b.begin(), b.end()), b.end());
        int first = b.empty() ? -1 : b[0];
        auto it = a.begin();
        while (it != a.end() && *it < first) ++it;
        a.insert(it, b.begin(), b.end());
        for (int i = 0; i < n; i++) cout << a[i] << " \n"[i + 1 == n];
    }
    return 0;
}
```
