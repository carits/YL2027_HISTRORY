# P0170 眼红的Medusa 题解

## 一、题意简化

本题可以简化为：

```text
给出获得科技创新奖的 n 个编号和获得特殊贡献奖的 m 个编号。输出同时获得两个奖项的人，顺序按科技创新奖名单中的先后顺序。
```

## 二、算法设计

把第二个名单排序，然后按第一个名单原顺序逐个二分查找，找到就输出。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `二分` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

O(m log m+n log m)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
