# P0198 Equalize by Divide 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 构造/贪心 |
| 子标签 | 操作构造, 最小值引导, 上界控制 |
| 三维难度 | 2/3/3 |
| 综合等级 | L3 |
| 推荐阶段 | 普及+ |
| 适合用途 | 构造操作题 |

## 训练价值

这题适合讲构造题里的“维护当前最小元素作为工具”。

## 课堂讲解建议

1. 先让学生复述题意中的限制条件。
2. 再引导学生抽象出核心状态或判定函数。
3. 对照样例手推一遍，确认边界情况。
4. 最后再讲代码结构和复杂度。

## 二刷建议

完成本题后，可以要求学生不看题解重新写一遍，并单独解释为什么算法不会漏情况。
