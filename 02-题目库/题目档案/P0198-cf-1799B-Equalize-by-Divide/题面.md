# P0198 Equalize by Divide 题面

## 问题陈述

给定正整数数组。一次操作可以选择 `i != j`，令 `a_i = ceil(a_i / a_j)`。判断是否能在不超过 `30n` 次操作内使所有数相等；若可以，输出任意一种操作方案。

## 输入格式

第一行输入测试组数。每组输入 $n$ 和数组。

## 输出格式

若无解输出 `-1`。否则先输出操作次数，再输出每次操作的两个下标 `i j`。

## 数据范围

`n <= 100`，若有解，题目保证存在不超过 `30n` 次的方案。

## 样例输入

```text
1
3
5 5 5
```

## 样例输出

```text
0
```
