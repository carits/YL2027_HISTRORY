# P0010 Following Directions 题面

## 来源

- 平台：CodeForces
- 题号：1791B
- 原题链接：https://codeforces.com/problemset/problem/1791/B
- VJudge 写法：`[problem:CodeForces-1791B]`

## 问题陈述

从坐标 (0,0) 出发，按照字符串中的 U、D、L、R 依次移动。判断移动过程中是否到达过坐标 (1,1)。

## 输入格式

第一行输入测试组数 T。每组先输入 n，再输入长度为 n 的移动字符串。

## 输出格式

每组输出 YES 或 NO。

## 自测样例

### 输入

```text
3
2
RU
4
DDDD
7
UURDDLL
```

### 输出

```text
YES
NO
YES
```
