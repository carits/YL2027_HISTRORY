# P0149 Equalizing by Division (hard version) 题面

## 题目来源

- 平台：CodeForces
- 题号：1213D2
- 原题链接：https://codeforces.com/problemset/problem/1213/D2
- VJudge 写法：`[problem:CodeForces-1213D2]`

## 问题陈述

给定 n 个整数。一次操作可以选择一个数并将它变成向下取整的一半。求最少操作次数，使数组中至少有 k 个数相等。

## 输入格式

第一行输入 n 和 k。第二行输入 n 个整数。

## 输出格式

输出最少操作次数。

## 约束要点

数值范围允许把每个数不断除以 2 并记录到达每个目标值的代价。

## 样例输入

```text
5 3
1 2 2 4 8
```

## 样例输出

```text
1
```
