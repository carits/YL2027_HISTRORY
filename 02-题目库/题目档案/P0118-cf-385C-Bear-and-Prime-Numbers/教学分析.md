# P0118 Bear and Prime Numbers 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 数论 |
| 子标签 | 素数筛, 倍数统计, 前缀和 |
| 三维难度 | 4/4/3 |
| 综合等级 | L5 |
| 推荐阶段 | 提高- |
| 适合用途 | 筛法综合题, 查询预处理题 |

## 训练价值

适合放在筛法专题后半段，重点训练“按质数枚举倍数”和“离线预处理回答多询问”。

## 讲评重点

1. 先让学生复述题意和限制条件。
2. 明确核心状态、枚举对象或判定函数。
3. 单独讲边界和常见错误。
4. 最后再给代码，避免学生只背实现。
