# P0048 第 k 小数

## 题目描述

给定长为$n$的数列$a_i$，由于长度较大，数列采用递推的方式生成。

给定$x、y$，假定$a_0 = 1$，$a_{i + 1} = (a_i * x + y) \mod (10^9 + 7)$，其中$a_0$不包含在数列中。

求数列中第$k$小的数。

## 输入格式

只有一行，包含四个整数$n$、$k$、$x$、$y$。

## 输出格式

只有一个整数。

## 说明与数据范围

对于30%的数据，$1 ≤ n ≤ 10000$。

对于100%的数据，$1 ≤ k ≤ n ≤ 10^7$，$1 ≤ x、y ≤ 10^9$。

## 样例

### 样例 1

**输入**

```text
10 4 2 3
```text

**输出**
```text
61
```text

### 样例 2

**输入**
```text
100 50 123 456
```text

**输出**
```text
389582539
```
