# P0159 【青苗杯2023小学组】比赛实况 题解

## 一、题意简化

本题可以归入 **数据结构**。核心任务是：根据输入限制，选择合适的数据结构或算法，避免直接写出超出复杂度的暴力做法。

## 二、思路分析

维护每名选手当前得分，并用有序 `map` 维护每种分数出现次数。每次加分时删除旧分数、加入新分数，最大值为 `rbegin`，最小值为 `begin`。

## 三、算法步骤

1. 读入题目数据。
2. 根据题意维护必要的状态或预处理信息。
3. 按照上述思路完成计算。
4. 输出题目要求的答案，注意空格、换行和特殊输出格式。

## 四、正确性说明

算法直接围绕题目中的不变量或等价转化进行：每一步维护的状态都与原问题中的真实状态一一对应；当状态更新结束时，得到的就是题目要求的最终结果。对于排序、枚举、模拟、动态规划或数学转化类题目，代码中的比较规则、转移规则或公式均来自题意本身，因此不会改变答案含义。

## 五、复杂度分析

$O(m\log n)$。

## 六、参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    cin >> n >> m;
    vector<long long> score(n + 1, 0);
    map<long long, int> cnt;
    cnt[0] = n;
    for (int i = 0; i < m; i++) {
        int p;
        long long v;
        cin >> p >> v;
        long long old = score[p];
        if (--cnt[old] == 0) cnt.erase(old);
        score[p] += v;
        cnt[score[p]]++;
        if (i) cout << ' ';
        cout << cnt.rbegin()->first - cnt.begin()->first;
    }
    cout << '\n';
    return 0;
}
```
