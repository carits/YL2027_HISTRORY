# P0107 Rudolph and Christmas Tree 题解

## 一、题意简化

本题的核心是：有 n 个相同的等腰三角形，每个三角形的底边长度为 d，高为 h。

## 二、算法设计

单个三角形面积为 d*h/2。

若相邻两个三角形竖直距离 delta >= h，则不重叠。

若 delta < h，重叠部分仍是一个与原三角形相似的小三角形，高为 h-delta，底边按比例变为 d*(h-delta)/h。扣掉这个重叠面积即可。
