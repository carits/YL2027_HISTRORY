## 问题陈述

有 n 个相同的等腰三角形，每个三角形的底边长度为 d，高为 h。它们沿同一竖直方向放置，第 i 个三角形的竖直位置由 y_i 给出。

相邻三角形可能发生重叠。求所有三角形覆盖区域的总面积。

## 输入格式

第一行输入测试组数 T。

每组输入 n、d、h。

接下来输入 n 个整数 y_i。

## 输出格式

每组输出总面积，允许小数误差。

## 数据范围

y_i 按从小到大给出。

## 样例输入

```text
1
2 2 2
0 1
```

## 样例输出

```text
3.5000000000
```
