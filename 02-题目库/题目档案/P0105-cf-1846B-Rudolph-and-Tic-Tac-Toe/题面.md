## 问题陈述

给定一个 3×3 的井字棋棋盘，格子可能是 X、O、+ 或 .。

若某个非 . 字符在任意一行、一列或一条对角线上连续出现 3 次，则该字符获胜。

若没有获胜者，输出 DRAW。

## 输入格式

第一行输入测试组数 T。

每组输入 3 行，每行 3 个字符。

## 输出格式

每组输出获胜字符 X、O、+，或 DRAW。

## 数据范围

棋盘大小固定为 3×3。

## 样例输入

```text
3
XXX
OO.
...
.O.
.O.
.O.
+..
.O.
..X
```

## 样例输出

```text
X
O
DRAW
```
