# P0083 Unfair Game 题解

## 一、题意简化

本题可以归入 **组合数学**。核心任务是：根据输入限制，选择合适的数据结构或算法，避免直接写出超出复杂度的暴力做法。

## 二、思路分析

对一个数，最优步数只与最高位位置和二进制中 $1$ 的个数有关。由于 $n=2^d$，$1\sim n-1$ 可以看作 $d$ 位内任意非空子集。枚举最高位和 $1$ 的个数，用组合数统计步数超过 $k$ 的数，最后单独考虑 $n$。

## 五、复杂度分析

$O(30^2)$ 每组。

## 六、参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

long long C[31][31];

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    for (int i = 0; i <= 30; i++) {
        C[i][0] = C[i][i] = 1;
        for (int j = 1; j < i; j++) C[i][j] = C[i - 1][j] + C[i - 1][j - 1];
    }

    int T;
    cin >> T;
    while (T--) {
        long long n;
        int k;
        cin >> n >> k;
        int d = 0;
        while (n % 2 == 0) {
            n /= 2;
            d++;
        }
        long long ans = 0;
        for (int mx = 0; mx < d; mx++) {
            for (int cnt = 1; cnt <= mx + 1; cnt++) {
                if (mx + cnt > k) ans += C[mx][cnt - 1];
            }
        }
        if (d + 1 > k) ans++;
        cout << ans << '\n';
    }
    return 0;
}
```
