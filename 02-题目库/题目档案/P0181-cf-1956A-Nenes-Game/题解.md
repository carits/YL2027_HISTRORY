# P0181 Nene's Game 题解

## 一、题意简化

给定一个递增位置序列 `a`。对每个初始人数 `n`，按照游戏规则反复删除指定位置上的人。求最后剩余人数。

## 二、算法设计

只有第一个位置 `a_1` 会决定答案。若当前人数小于 `a_1`，本轮无人被删除，人数保持不变；若人数至少为 `a_1`，每轮至少会删掉第 `a_1` 个位置的人，人数最终降到 `a_1-1`。所以答案是 `min(n, a_1-1)`。

## 三、正确性说明

当人数小于 `a_1` 时，所有删除位置都越界，过程停止。当人数不小于 `a_1` 时，一轮后人数减少，重复后必然降到 `a_1-1`；一旦降到该值，又没有位置可删。

## 四、复杂度分析

每个询问 `O(1)`。
