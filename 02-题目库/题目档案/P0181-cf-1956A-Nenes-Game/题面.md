# P0181 Nene's Game 题面

## 问题陈述

给定一个递增位置序列 $a$。对每个初始人数 $n$，按照游戏规则反复删除指定位置上的人。求最后剩余人数。

## 输入格式

每组测试输入 `k, q`，随后输入 $k$ 个位置 $a_i$，再输入 $q$ 个询问人数 $n_i$。

## 输出格式

每组询问在一行输出答案。

## 数据范围

多组测试；位置序列递增。

## 样例输入

```text
1
3 4
4 7 9
1 3 5 10
```

## 样例输出

```text
1 3 3 3
```
