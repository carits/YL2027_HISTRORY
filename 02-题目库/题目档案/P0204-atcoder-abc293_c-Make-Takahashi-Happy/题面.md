# P0204 Make Takahashi Happy 题面

## 题目来源

- 平台：AtCoder
- 题号：abc293_c
- 原题链接：https://atcoder.jp/contests/abc293/tasks/abc293_c
- VJudge 写法：`[problem:AtCoder-abc293_c]`

## 问题描述

在 `H × W` 网格中，每个格子有一个整数。从左上角走到右下角，每步只能向右或向下。若路径经过的所有数字互不相同，则这条路径合法。求合法路径数量。

## 输入格式

输入 `H,W`，然后输入网格 `A`。

## 输出格式

输出合法路径数量。

## 数据范围

`2 <= H,W <= 10`，`1 <= A_{i,j} <= 10^9`。

## 样例输入

```text
3 3
3 2 2
2 1 3
1 5 4
```

## 样例输出

```text
3
```
