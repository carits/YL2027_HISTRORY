# P0002 陶陶摘苹果 题解

## 一、题意简化

本题核心可以归纳为：

```text
围绕 `枚举` 专题，完成 `身高比较, 计数`，并注意 `比较条件错误, 输入处理错误`。
```

## 二、算法设计

### 核心思路

- 主标签：枚举
- 子标签：数组遍历, 条件统计
- 关键技巧：身高比较, 计数

处理流程：

1. 读入题目数据。
2. 根据题意确定需要维护的状态或枚举对象。
3. 按照 `身高比较, 计数` 完成模拟、枚举或统计。
4. 输出最终答案。

## 三、复杂度分析

由于该题定位为 入门 阶段训练题，通常正解复杂度应满足原题限制。正式补全题面后，可在此处写明准确复杂度。

## 四、常见错误

- 比较条件错误, 输入处理错误
- 输入输出格式不严格。
- 边界条件没有单独检查。

## 五、教学讲法

这题适合围绕以下问题提问：

1. 题目要求我们维护什么？
2. 枚举对象是什么？
3. 是否存在容易漏掉的边界？
4. 代码中哪一部分最容易写错？
