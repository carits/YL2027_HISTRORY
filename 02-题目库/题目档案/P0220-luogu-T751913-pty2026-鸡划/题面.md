## 鸡划（plan）

#### 题目描述

胖头鱼在回到大海之前，还想邀请朋友一起去进行一次健身，胖头鱼有 $n$ 个朋友，朋友的编号为 $1 \dots n$ ，为了方便，他打算邀请一段**编号连续**的朋友。

朋友在一起健身时，总是会聊一些爱好，一共有 $m$ 种爱好，爱好的编号为 $1$ 到 $m$ ，第 $i$ 个朋友拥有 $c_i$ 个爱好，分别为 $d_{i, 1}, \dots, d_{i, c_i}$ 。

两个朋友如果**没有任何共同**的爱好，他们在健身时就不会进行交流。如果有至少 $k$ 对朋友之间不会交流，那么这次健身的气氛就会变得有些尴尬。现在胖头鱼想知道，在保证健身氛围不尴尬的前提下，**最多**能够邀请多少个朋友，由于他只擅长健身，所以只好请你帮忙计算一下。

#### 输入格式（文件名：plan.in）

第一行包含三个整数 $n, m, k$ ，表示朋友的数量、爱好的数量和使得气氛变得尴尬的最少的不交流的朋友对的数量。

接下来 $n$ 行，每行首先包含一个整数 $c_i$ ，表示第 $i$ 个朋友拥有的爱好数量；然后包含 $c_i$ 个**互不相同**的整数 $d_{i, j}$ ，表示每个爱好的编号。

#### 输出格式（文件名：plan.out）

输出一个整数，表示最多可以邀请的朋友数量。

#### 样例数据

存放在 `sample/plan` 目录下。

【样例 $1$ 输入】

存放在 `plan1.in` 文件中。

```
10 1 6
1 1
1 1
0
1 1
1 1
0
1 1
1 1
1 1
1 1
```

【样例 $1$ 输出】

存放在 `plan1.out` 文件中。

```
6
```

【样例 $1$ 解释】

可以邀请编号 $4$ 到编号 $9$ 的朋友，只有编号为 $6$ 的朋友不会与其他人交流，其他朋友互相之间都会交流，所以共有 $5$ 对朋友之间不会交流。

【样例 $2$ 输入】

存放在 `plan2.in` 文件中。

```
10 2 2
0
1 1 
2 1 2
1 2
0
2 1 2
1 2
1 1
0
2 1 2
```

【样例 $2$ 输出】

存放在 `plan2.out` 文件中。

```
3
```

【样例 $2$ 解释】

可以邀请编号 $2$ 到编号 $4$ 的朋友，只有编号 $2$ 和编号 $4$ 的朋友不会交流。

【样例 $3, 4, 5$ 输入与输出】

存放在 `plan3.in` 与 `plan3.out` 、 `plan4.in` 与 `plan4.out` 、 `plan5.in` 与 `plan5.out` 文件中。

#### 数据说明

对于所有测试点，满足 $1 \le n \le 10^5$ 、 $1 \le d_{i, j} \le m \le 10$ 、 $0 \le c_i \le m$ 、 $1 \le k \le n^2$ 。

测试点的特殊属性如下：
  - 对于前 $15\%$ 的测试点，满足 $m = k = 1$ 。
  - 对于额外 $15\%$ 的测试点，满足 $n \le 100$ 。
  - 对于额外 $15\%$ 的测试点，满足 $n \le 1000$ 且 $m = 1$ 。样例 $3$ 符合当前性质。
  - 对于额外 $10\%$ 的测试点，满足 $n \le 1000$ 且 $m = 2$ 。
  - 对于额外 $10\%$ 的测试点，满足 $n \le 1000$ 且 $m \le 6$ 。样例 $4$ 符合当前性质。
  - 对于额外 $15\%$ 的测试点，满足 $m = 1$ 。
  - 对于额外 $20\%$ 的测试点，没有特殊属性。样例 $5$ 符合当前性质。
