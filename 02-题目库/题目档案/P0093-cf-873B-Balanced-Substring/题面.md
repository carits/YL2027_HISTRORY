## 问题陈述

给定一个长度为 n 的 01 字符串。

求最长的连续子串长度，使得子串中 0 和 1 的数量相同。

## 输入格式

第一行输入 n。

第二行输入一个长度为 n 的 01 字符串。

## 输出格式

输出满足条件的最长子串长度。

## 数据范围

1 <= n <= 100000。

## 样例输入

```text
8
11001010
```

## 样例输出

```text
8
```
