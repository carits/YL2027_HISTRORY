# P0093 Balanced Substring 题解

## 一、题意简化

本题的核心是：给定一个长度为 n 的 01 字符串。

## 二、算法设计

把 1 看成 +1，把 0 看成 -1。

若两个位置的前缀和相同，则它们之间这一段的总和为 0，也就是 0 和 1 数量相同。

记录每个前缀和第一次出现的位置，后面再遇到同样前缀和时更新答案。

## 三、正确性说明

算法直接围绕题目要求维护必要状态。每次转移或统计都只使用已经确定的信息，并且覆盖所有可能答案；最终取到的结果与题目定义一致。

## 四、复杂度分析

时间复杂度 O(n)，空间复杂度 O(n)。
