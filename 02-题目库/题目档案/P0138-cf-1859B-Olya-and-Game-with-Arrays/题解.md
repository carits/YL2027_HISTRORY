# P0138 Olya and Game with Arrays 题解

## 一、题意简化

本题需要围绕 **贪心**，处理 `最小值, 次小值, 数组重组` 相关问题。

## 二、算法设计

每个数组排序后，若拿走最小值，这个数组新的最小值会变成次小值。

最终只有一个数组需要承接所有被拿走的最小值，因此答案为：所有数组次小值之和，减去最小的次小值，再加上全局最小值。

## 三、正确性说明

算法的核心是抓住题目中的不变量或最优选择依据，并在每一步只保留对后续有意义的信息。对于需要枚举的题目，枚举对象被限制在可证明的有限范围内；对于贪心题，选择依据保证不会让后续状态变差；对于动态规划题，状态转移完整覆盖所有可行路径。

## 四、复杂度分析

对每个数组排序，复杂度为输入规模排序复杂度。

## 五、常见错误

- 没有单独处理边界输入。
- 使用 `int` 导致中间结果溢出。
- 枚举范围过大或过小。
- 多组数据时没有清空临时变量。
