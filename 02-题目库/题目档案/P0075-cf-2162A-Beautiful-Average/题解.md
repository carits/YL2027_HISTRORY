# P0075 Beautiful Average 题解

## 一、题意简化

本题可以简化为：

```text
给定数组，求任意非空连续子数组平均值的最大可能值。
```

## 二、算法设计

任意一段的平均值不可能超过这一段内的最大元素，而选择单个最大元素作为子数组即可达到该值，所以答案就是数组最大值。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `贪心` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

每组 O(n)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
