# P0169 烦恼的高考志愿 题面

## 题目来源

- 平台：洛谷
- 题号：P1678
- 原题链接：https://www.luogu.com.cn/problem/P1678
- VJudge 写法：`[problem:洛谷-P1678]`

## 问题陈述

给定若干学校分数线和若干学生估分。每个学生会选择与自己估分差距最小的学校，贡献为这个最小差值。求所有学生贡献之和。

## 输入格式

第一行输入 $m$ 和 $n$。第二行输入 m 个学校分数线。第三行输入 n 个学生估分。

## 输出格式

输出差值之和。

## 约束要点

需要排序后对每个学生二分最近分数线。

## 样例输入

```text
4 3
500 600 700 800
550 650 900
```

## 样例输出

```text
200
```
