# P0110 Lunatic Never Content 题解

## 一、题意简化

本题需要围绕 **数学**，处理 `最大公约数, 回文差值` 相关问题。

## 二、算法设计

如果替换后要成为回文，则对于每一对对称位置 $i$ 和 $n$-i+1，有：

a_i mod x = a_j mod x。

这等价于 x 整除 |a_i - a_j|。

所以答案就是所有对称差值的 $\gcd$。若数组本来对称，所有差值都是 0，gc$d$ 为 0，表示任意 x 都可行。

## 三、正确性说明

算法的核心是抓住题目中的不变量或最优选择依据，并在每一步只保留对后续有意义的信息。对于需要枚举的题目，枚举对象被限制在可证明的有限范围内；对于贪心题，选择依据保证不会让后续状态变差；对于动态规划题，状态转移完整覆盖所有可行路径。

## 四、复杂度分析

每组 O(n log V)。

## 五、常见错误

- 没有单独处理边界输入。
- 使用 `int` 导致中间结果溢出。
- 枚举范围过大或过小。
- 多组数据时没有清空临时变量。
