## 问题陈述

给定长度为 n 的二进制串 s，以及整数 $x$ 和 $y$。只关心最后 x 位：其中从右往左数第 y 位必须为 1，最后 x 位中的其他位置必须为 0。

每次可以翻转一个字符，求最少翻转次数。

## 输入格式

输入 n、x、y。

再输入二进制字符串 s。

## 输出格式

输出最少翻转次数。

## 数据范围

0 <= y < x <= n。

## 样例输入

```text
5 3 1
10101
```

## 样例输出

```text
3
```
