# P0061 Remainder 题解

## 一、题意简化

给定长度为 n 的二进制串 s，以及整数 $x$ 和 $y$。只关心最后 x 位：其中从右往左数第 y 位必须为 1，最后 x 位中的其他位置必须为 0。

## 二、算法设计

最后 x 位的下标范围是 [n-x, n-1]。

其中目标为 1 的位置是 n-1-y，其余位置目标为 0。逐位比较当前字符和目标字符，统计不同的数量。

## 三、复杂度分析

时间复杂度 O(x)。

## 四、易错点

- 没有严格按照多组数据格式读取。
- 下标从 0 开始和从 1 开始混用。
- 只过了样例，没有检查边界情况。
- 输出格式多空格或少换行。
