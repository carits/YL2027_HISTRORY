# P0007 铺地毯 题面

## 来源

- 平台：洛谷
- 题号：P1003
- 原题链接：https://www.luogu.com.cn/problem/P1003
- VJudge 写法：`[problem:洛谷-P1003]`

## 问题陈述

按顺序铺 n 张矩形地毯。每张地毯给出左下角坐标 (x,y) 以及横向长度 a、纵向长度 b，覆盖范围为 x<=X<=x+a 且 y<=Y<=y+b。给出一个查询点，求这个点最上面被哪一张地毯覆盖；若没有地毯覆盖，输出 -1。

## 输入格式

第一行输入整数 n。
接下来 $n$ 行每行输入 x, y, a, b，表示一张地毯。
最后一行输入查询点坐标 X, Y。

## 输出格式

输出覆盖查询点的最上层地毯编号；若没有覆盖，输出 -1。

## 自测样例

### 输入

```text
3
1 0 2 3
0 2 3 3
2 1 3 3
2 2
```

### 输出

```text
3
```
