# P0190 a-Good String 题解

## 一、题意简化

长度为 `2^k` 的字符串可以按递归规则定义为 `c-good`：要么长度为 1 且等于字符 `c`，要么一半全是 `c`，另一半是 `(c+1)-good`。给定字符串，求把它改成 `a-good` 的最少修改次数。

## 二、算法设计

递归处理区间 `[l,r]` 和当前目标字符 `c`。有两种选择：左半全改成 `c`，右半递归变成 `c+1-good`；或者右半全改成 `c`，左半递归。取两者最小。

## 三、正确性说明

定义本身只允许这两种递归结构，因此最优解必在两种选择之一中。全改一半的代价由该半区不等于 `c` 的字符数决定，另一半继续满足子问题。递归取最小覆盖全部合法结构。

## 四、复杂度分析

时间复杂度 `O(n log n)` 或用前缀计数优化到 `O(26n)`；本实现直接计数，总数据范围内可通过。
