# P0190 a-Good String 题面

## 问题陈述

长度为 `2^k` 的字符串可以按递归规则定义为 $c-good$：要么长度为 1 且等于字符 $c$，要么一半全是 $c$，另一半是 `(c+1)-good`。给定字符串，求把它改成 $a-good$ 的最少修改次数。

## 输入格式

第一行输入测试组数。每组输入 $n$ 和字符串 $s$。

## 输出格式

每组输出最少修改次数。

## 数据范围

$n$ 是 2 的幂，所有测试的 $n$ 总和通常不超过 $200000$。

## 样例输入

```text
2
1
z
2
ac
```

## 样例输出

```text
1
1
```
