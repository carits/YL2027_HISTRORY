# P0065 Reverse a Permutation 题解

## 一、题意简化

本题可以简化为：

```text
给定一个 1 到 n 的排列。必须恰好选择一个区间并将其反转。求所有可能结果中字典序最大的排列。
```

## 二、算法设计

从左到右看，若当前位置已经是当前后缀能放到这里的最大值，就不需要动它；遇到第一个不满足的位置，就把该后缀最大值所在位置反转到这里。若排列已经完全降序，反转长度为 1 的区间即可保持不变。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `贪心` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

每组 O(n)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
