# P0134 Don't Try to Count 题面

## 题目来源

- 平台：CodeForces
- 题号：1881A
- 原题链接：https://codeforces.com/problemset/problem/1881/A
- VJudge 写法：`[problem:CodeForces-1881A]`

## 问题陈述

给定字符串 x 和 s。每次操作可以把 x 变成 x + x。

求最少操作次数，使得 s 成为 x 的子串。若无法做到，输出 -1。

## 输入格式

第一行输入 T。

每组输入 n、m，再输入字符串 x 和 s。

## 输出格式

每组输出最少操作次数或 -1。

## 数据范围

n、m 为字符串长度。

## 样例输入

```text
3
3 5
abc
bcabc
3 3
abc
abc
1 5
a
aaaaa
```

## 样例输出

```text
1
0
3
```
