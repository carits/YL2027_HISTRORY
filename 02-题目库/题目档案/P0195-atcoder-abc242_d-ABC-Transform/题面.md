# P0195 ABC Transform 题面

## 问题陈述

字符串由 `A,B,C` 组成。每次变换中，`A` 变为 `BC`，`B` 变为 `CA`，`C` 变为 `AB`。给定初始字符串和多组询问 `(t,k)`，求经过 `t` 次变换后第 `k` 个字符。

## 输入格式

输入字符串 `S`，再输入询问数 `Q`。每个询问输入 `t, k`。

## 输出格式

每个询问输出一个字符。

## 数据范围

`t` 和 `k` 可很大，需要按祖先反推，不能展开字符串。

## 样例输入

```text
ABC
3
1 1
1 2
2 3
```

## 样例输出

```text
B
C
A
```
