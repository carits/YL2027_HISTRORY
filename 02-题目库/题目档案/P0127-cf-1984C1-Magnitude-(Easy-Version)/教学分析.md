# P0127 Magnitude (Easy Version) 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 动态规划 |
| 子标签 | 状态极值, 绝对值操作, 贪心DP |
| 三维难度 | 2/3/2 |
| 综合等级 | L3 |
| 推荐阶段 | 普及+ |
| 适合用途 | DP入门提升题 |

## 训练价值

适合讲“有时最小值也是未来最大值的来源”，帮助学生理解极值 DP。

## 讲评重点

1. 先让学生复述题意和限制条件。
2. 明确核心状态、枚举对象或判定函数。
3. 单独讲边界和常见错误。
4. 最后再给代码，避免学生只背实现。
