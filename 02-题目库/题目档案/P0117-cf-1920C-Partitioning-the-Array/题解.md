# P0117 Partitioning the Array 题解

## 一、题意简化

本题需要围绕 **数学**，处理 `因子枚举, gcd, 周期性质` 相关问题。

## 二、算法设计

固定 k 时，同一组内任意相邻两个元素 a_i 和 a_{i+k} 的差值都必须被同一个模数整除。

因此计算所有 |a_i - a_{i+k}| 的 gcd。若 gcd 不等于 1，则合法；gcd 为 0 表示这些位置完全相等，也合法。

只需要枚举 n 的因子 k。

## 三、正确性说明

算法的核心是抓住题目中的不变量或最优选择依据，并在每一步只保留对后续有意义的信息。对于需要枚举的题目，枚举对象被限制在可证明的有限范围内；对于贪心题，选择依据保证不会让后续状态变差；对于动态规划题，状态转移完整覆盖所有可行路径。

## 四、复杂度分析

枚举所有因子并扫描，整体约 O(n * 因子数)。

## 五、常见错误

- 没有单独处理边界输入。
- 使用 `int` 导致中间结果溢出。
- 枚举范围过大或过小。
- 多组数据时没有清空临时变量。
