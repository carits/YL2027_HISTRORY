# P0108 Rudolph and Snowflakes (simple version) 题解

## 一、题意简化

本题的核心是：给定整数 n，判断是否存在整数 k >= 2 和至少 3 项，使得：

n = 1 + k + k^2 + ... + k^m，其中 m >= 2。

## 二、算法设计

枚举底数 k。最短合法形式是 1+k+k^2，因此只需要枚举满足 1+k+k^2 <= n 的 k。

对固定 k，不断累加等比数列，若某一刻等于 n，则答案为 YES；若超过 n，则换下一个 k。
