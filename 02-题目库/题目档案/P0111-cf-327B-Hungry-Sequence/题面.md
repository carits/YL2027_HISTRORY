# P0111 Hungry Sequence 题面

## 题目来源

- 平台：CodeForces
- 题号：327B
- 原题链接：https://codeforces.com/problemset/problem/327/B
- VJudge 写法：`[problem:CodeForces-327B]`

## 问题描述

构造一个长度为 $n$ 的严格递增正整数序列，所有元素不超过 `10^7`，并且对于任意 `i < j`，$a_j$ 不能被 $a_i$ 整除。输出任意一种合法序列。

## 输入格式

输入一个整数 $n$。

## 输出格式

输出 $n$ 个整数，表示一组合法序列。

## 数据范围

`1 <= n <= 100000`。

## 样例输入

```text
3
```

## 样例输出

```text
4 5 6
```
