# P0111 Hungry Sequence 题解

## 一、题意简化

构造一个长度为 `n` 的严格递增正整数序列，所有元素不超过 `10^7`，并且对于任意 `i < j`，`a_j` 不能被 `a_i` 整除。输出任意一种合法序列。

## 二、算法设计

输出 `n + 1, n + 2, ..., 2n`。这个序列严格递增，并且任意前面的数都大于 `n`，后面的数不超过 `2n`。

## 三、正确性说明

若 `i < j`，则 `a_i >= n+1` 且 `a_j <= 2n`，所以 `2a_i > 2n >= a_j`。因此 `a_j` 不可能是 `a_i` 的两倍或更大倍数，又因为二者不同，所以不存在整除关系。

## 四、复杂度分析

时间复杂度 `O(n)`，空间复杂度 `O(1)`。
