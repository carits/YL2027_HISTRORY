# P0211 特殊的质数肋骨 题面

## 题目来源

- 平台：洛谷
- 题号：P1218
- 原题链接：https://www.luogu.com.cn/problem/P1218
- VJudge 写法：`[problem:洛谷-P1218]`

## 问题描述

求所有长度为 $n$ 的特殊质数：从右侧不断删去数字后，每次剩下的前缀仍然是质数。例如 $7331$、$733$、$73$、$7$ 都是质数，所以 $7331$ 是特殊质数。

## 输入格式

输入一个整数 $n$。

## 输出格式

按从小到大的顺序，每行输出一个长度为 $n$ 的特殊质数。

## 数据范围

`1 <= n <= 8`。

## 样例输入

```text
2
```

## 样例输出

```text
23
29
31
37
53
59
71
73
79
```
