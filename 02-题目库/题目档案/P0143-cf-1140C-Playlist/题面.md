# P0143 Playlist 题面

## 题目来源

- 平台：CodeForces
- 题号：1140C
- 原题链接：https://codeforces.com/problemset/problem/1140/C
- VJudge 写法：`[problem:CodeForces-1140C]`

## 问题陈述

有 n 首歌，每首歌有时长 t_i 和美丽值 b_i。你最多可以选择 k 首歌。一个歌单的快乐值定义为：所选歌曲总时长乘以所选歌曲中的最小美丽值。求快乐值最大是多少。

## 输入格式

第一行输入 n 和 k。接下来 n 行，每行输入 t_i 和 b_i。

## 输出格式

输出最大快乐值。

## 约束要点

n 较大，需要 O(n log n) 级别算法；答案可能超过 32 位整数。

## 样例输入

```text
3 2
4 7
15 1
3 6
```

## 样例输出

```text
42
```
