# P0193 Equivalent Strings 题面

## 问题陈述

两个等长字符串被称为等价：要么它们完全相同；要么长度为偶数时，将两个字符串分别分成左右两半，左右对应等价或交叉对应等价。给定两个字符串，判断它们是否等价。

## 输入格式

输入两个等长小写字符串。

## 输出格式

若等价输出 `YES`，否则输出 `NO`。

## 数据范围

字符串长度最多约 $200000$。

## 样例输入

```text
aaba
abaa
```

## 样例输出

```text
YES
```
