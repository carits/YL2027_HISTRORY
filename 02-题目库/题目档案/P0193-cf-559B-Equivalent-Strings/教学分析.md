# P0193 Equivalent Strings 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 字符串/分治 |
| 子标签 | 规范化, 等价关系, 递归排序 |
| 三维难度 | 2/4/2 |
| 综合等级 | L4 |
| 推荐阶段 | 提高- |
| 适合用途 | 字符串分治题 |

## 训练价值

可以作为“递归定义的等价类规范化”例题，和哈希优化版本对比讲。

## 课堂讲解建议

1. 先让学生复述题意中的限制条件。
2. 再引导学生抽象出核心状态或判定函数。
3. 对照样例手推一遍，确认边界情况。
4. 最后再讲代码结构和复杂度。

## 二刷建议

完成本题后，可以要求学生不看题解重新写一遍，并单独解释为什么算法不会漏情况。
