# P0019 双数最大和

## 题目描述

给定$n$个数组成的正整数有序数列$a_i$，要求选取两个整数，总和不超过$t$，求最大总和。

无解则输出`0`。

## 输入格式

第一行包含整数$n$、$t$。

第二行包含$n$个正整数$a_i$。

## 输出格式

只有一行，包含最大总和。

## 说明与数据范围

对于50%的数据，$1 ≤ n ≤ 1000$。

对于100%的数据，$1 ≤ n ≤ 10^5$，$1 ≤ a_i ≤ 10^9$。

## 样例

### 样例 1

**输入**

```text
5 8
1 3 4 6 9
```text

**输出**
```text
7
```
