## 问题陈述

给定一个整数数组。

统计有多少个偶数长度的连续子数组，其整体异或和为 0。

## 输入格式

第一行输入 n。

第二行输入 n 个整数 a_i。

## 输出格式

输出满足条件的子数组个数。

## 数据范围

1 <= n <= 300000。

## 样例输入

```text
3
1 1 1
```

## 样例输出

```text
2
```
