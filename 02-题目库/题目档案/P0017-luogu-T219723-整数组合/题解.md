# P0017 整数组合 题解

## 一、题意简化

## 二、思路分析

这是只有三种物品的完全背包最小件数问题。设 $dp[s]$ 表示凑出总和 $s$ 的最少个数，从 $0$ 出发向 $s+a,s+b,s+c$ 转移。

## 五、复杂度分析

$O(n)$。

## 六、参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, a, b, c;
    cin >> n >> a >> b >> c;
    const int inf = 1e9;
    vector<int> dp(n + 1, inf);
    dp[0] = 0;
    for (int i = 0; i <= n; i++) {
        if (dp[i] == inf) continue;
        if (i + a <= n) dp[i + a] = min(dp[i + a], dp[i] + 1);
        if (i + b <= n) dp[i + b] = min(dp[i + b], dp[i] + 1);
        if (i + c <= n) dp[i + c] = min(dp[i + c], dp[i] + 1);
    }
    cout << (dp[n] == inf ? -1 : dp[n]) << '\n';
    return 0;
}
```
