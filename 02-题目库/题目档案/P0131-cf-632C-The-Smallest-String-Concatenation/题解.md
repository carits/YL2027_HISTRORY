# P0131 The Smallest String Concatenation 题解

## 一、题意简化

本题需要围绕 **排序**，处理 `自定义比较, 字符串拼接` 相关问题。

## 二、算法设计

排序时不能只比较单个字符串大小。

对于两个字符串 a 和 b，如果 a+b < b+a，则 a 应排在 b 前面。按这个比较器排序后整体拼接即为最优。

## 三、正确性说明

算法的核心是抓住题目中的不变量或最优选择依据，并在每一步只保留对后续有意义的信息。对于需要枚举的题目，枚举对象被限制在可证明的有限范围内；对于贪心题，选择依据保证不会让后续状态变差；对于动态规划题，状态转移完整覆盖所有可行路径。

## 四、复杂度分析

排序 O(n log n)，比较代价与字符串长度有关。

## 五、常见错误

- 没有单独处理边界输入。
- 使用 `int` 导致中间结果溢出。
- 枚举范围过大或过小。
- 多组数据时没有清空临时变量。
