# P0131 The Smallest String Concatenation 题面

## 题目来源

- 平台：CodeForces
- 题号：632C
- 原题链接：https://codeforces.com/problemset/problem/632/C
- VJudge 写法：`[problem:CodeForces-632C]`

## 问题陈述

给定 $n$ 个字符串。你可以任意排列它们，然后按顺序拼接成一个大字符串。

求字典序最小的拼接结果。

## 输入格式

第一行输入 n。

接下来 $n$ 行每行一个字符串。

## 输出格式

输出字典序最小的拼接字符串。

## 数据范围

1 <= n，字符串非空。

## 样例输入

```text
3
c
cb
cba
```

## 样例输出

```text
cbacbc
```
