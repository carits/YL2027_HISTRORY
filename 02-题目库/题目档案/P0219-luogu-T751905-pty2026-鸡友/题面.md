## 鸡友（friends）

#### 题目描述

胖头鱼打算拜访 $n$ 个朋友，朋友的编号为 $1 \dots n$ ，为了方便，他每天会拜访一段**连续编号**的朋友。第 $i$ 个朋友的热情度为 $a_i$ ，胖头鱼每天获得的热情度就是拜访的**所有**朋友的热情度的**总和**。胖头鱼计划用 $k$ 天将每个朋友都**刚好**拜访一次，虽然他不会数论，但是他最近学会了一个很酷的词汇叫做**最大公约数**，所以他希望这 $k$ 天获得的热情度的最大公约数尽可能大。

由于他还不确定需要花几天来拜访朋友，所以请对 $1$ 到 $n$ 中的每个整数 $k$ ，计算每天热情度的最大公约数的**最大值**。

#### 输入格式（文件名：friends.in）

第一行包含一个整数 $n$ ，表示朋友的数量。

第二行包含 $n$ 个整数 $a_i$ ，第 $i$ 个整数表示第 $i$ 个朋友的热情度。

#### 输出格式（文件名：friends.out）

只有一行，输出 $n$ 个整数，之间用一个空格隔开。第 $i$ 个整数表示当胖头鱼计划用 $i$ 天拜访完所有朋友时，每天热情度的最大公约数的最大值。

#### 样例数据

存放在 `sample/friends` 目录下。

【样例 $1$ 输入】

存放在 `friends1.in` 文件中。

```
6
1 1 1 1 1 1
```

【样例 $1$ 输出】

存放在 `friends1.out` 文件中。

```
6 3 2 1 1 1
```

【样例 $1$ 解释】

用 $1$ 天时，热情度为 $6$ ；用 $2$ 天时，可以每天拜访 $3$ 个朋友，每天的热情度均为 $3$ ；用 $3$ 天时，可以每天拜访 $2$ 个朋友，每天的热情度均为 $2$ ；用不少于 $4$ 天时，至少有 $1$ 天的热情度为 $1$ 。

【样例 $2$ 输入】

存放在 `friends2.in` 文件中。

```
6
1 2 3 4 5 6
```

【样例 $2$ 输出】

存放在 `friends2.out` 文件中。

```
21 3 3 3 1 1
```

【样例 $2$ 解释】

用 $2$ 天时，可以每天分别拜访 $5, 1$ 个朋友，热情度为 $15, 6$ ；用 $3$ 天时，可以每天分别拜访 $3, 2, 1$ 个朋友，热情度为 $6, 9, 6$ ；用 $4$ 天时，可以每天分别拜访 $2, 1, 2, 1$ 个朋友，热情度为 $3, 3, 9, 6$ 。

【样例 $3, 4, 5$ 输入与输出】

存放在 `friends3.in` 与 `friends3.out` 、 `friends4.in` 与 `friends4.out` 、 `friends5.in` 与 `friends5.out` 文件中。

#### 数据说明

对于所有测试点，满足 $1 \le n \le 10^5$ 、 $1 \le a_i \le 10^7$ 。

测试点的特殊属性如下：
  - 对于前 $25\%$ 的测试点，满足 $n \le 20$ 。
  - 对于额外 $25\%$ 的测试点，满足 $n \le 5000$ 且 $a_i = 1$ 。样例 $3$ 符合当前性质。
  - 对于额外 $30\%$ 的测试点，满足 $n, a_i \le 10000$ 。样例 $4$ 符合当前性质。
  - 对于额外 $20\%$ 的测试点，没有特殊属性。样例 $5$ 符合当前性质。
