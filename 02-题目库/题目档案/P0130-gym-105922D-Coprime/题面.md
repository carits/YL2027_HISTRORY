# P0130 Coprime

## 题目描述

给定两个正整数 $x,y$，请找一个正整数 $z$，满足：

- $x < z < y$；
- $\gcd(z,x)=1$ 且 $\gcd(z,y)=1$。

如果有多个答案，输出任意一个；如果不存在合法的 $z$，输出 `-1`。

## 输入格式

第一行包含整数 $T$，表示测试数据组数。

接下来 $T$ 行，每行包含两个整数 $x,y$。

## 输出格式

对每组数据，若存在合法的 $z$，输出任意一个合法值；否则输出 `-1`。

## 数据范围

$1 \le T \le 10^4$，$1 \le x,y \le 10^{10}$，$x+2 \le y$。

## 样例

### 样例 1

**输入**

```text
3
1 3
115 125
2184 2200
```

**输出**

```text
2
117
-1
```

## 样例说明

第二组数据中，`117` 与 `115`、`125` 均互质，因此是合法答案之一。
