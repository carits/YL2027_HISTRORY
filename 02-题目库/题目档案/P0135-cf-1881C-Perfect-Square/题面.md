# P0135 Perfect Square 题面

## 题目来源

- 平台：CodeForces
- 题号：1881C
- 原题链接：https://codeforces.com/problemset/problem/1881/C
- VJudge 写法：`[problem:CodeForces-1881C]`

## 问题陈述

给定 $n$ x n 的小写字母矩阵。一次操作可以把某个字符变成字母表中后面的字符，代价为前进的步数。

要求把矩阵变成旋转 90 度后仍然相同的矩阵，求最小总代价。

## 输入格式

第一行输入 T。

每组输入 n，随后输入 n 行字符串表示矩阵。

## 输出格式

每组输出最小总代价。

## 数据范围

$n$ 为正整数。

## 样例输入

```text
2
2
ab
cd
3
aaa
aaa
aaa
```

## 样例输出

```text
6
0
```
