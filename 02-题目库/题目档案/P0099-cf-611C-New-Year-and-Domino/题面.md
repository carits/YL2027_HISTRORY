## 问题陈述

给定一个 n*m 的网格，'.' 表示空格，'#' 表示障碍。

每次询问一个矩形区域，要求统计在这个矩形内可以放置一个 1×2 或 2×1 多米诺骨牌的位置数。骨牌必须覆盖两个相邻空格，并且完全位于询问矩形内。

## 输入格式

第一行输入 n 和 m。

接下来 n 行输入网格。

之后输入 q，再输入 q 个询问 x1 y1 x2 y2。

## 输出格式

对每个询问输出可放置位置数量。

## 数据范围

1 <= n, m <= 500。

## 样例输入

```text
3 3
...
.#.
...
3
1 1 3 3
1 1 1 3
2 1 3 1
```

## 样例输出

```text
8
2
1
```

样例用于说明输入输出格式与核心计算过程，讲评时可结合题解手推。
