# P0009 Yes-Yes? 教学分析

## 一、教学定位

| 项目 | 内容 |
|---|---|
| 推荐阶段 | 入门 |
| 综合等级 | L1 |
| 三维难度 | 1/1/1 |
| 适合用途 | 入门题, 适合作业 |
| 前置知识 | 字符串基础 |

## 二、训练价值

- 主要训练：训练构造长串后判断包含关系
- 知识点：字符串 / 字符串匹配, 子串判断
- 能力标签：读题理解, 模型转化
- 常见错误：大小写错误, 子串判断错误

## 三、课堂讲解重点

1. 先让学生复述题意，确认输入输出目标。
2. 明确本题的枚举对象或状态变化。
3. 单独强调边界条件。
4. 让学生估计复杂度，避免无意义暴力。
5. 对照样例手推一遍。

## 四、学生常见表现

| 表现 | 说明 | 处理方式 |
|---|---|---|
| 能看懂题但不会写循环 | 基础实现不稳 | 要求手写流程表 |
| 代码能过样例但边界挂 | 边界意识不足 | 增加极端样例检查 |
| 思路正确但变量混乱 | 实现稳定性不足 | 规范变量含义和初始化 |

## 五、讲评建议

- 讲评时不要只给代码，应先问“本题到底在维护什么”。
- 对易错题要补一个反例或边界样例。
- 对 L2-L3 题目建议安排一次二刷。
