# P0009 Yes-Yes? 题面

## 来源

- 平台：CodeForces
- 题号：1759A
- 原题链接：https://codeforces.com/problemset/problem/1759/A

## 问题陈述

给出一个字符串 s，判断它是否可以作为无限字符串 YesYesYesYes... 的一个连续子串。

## 输入格式

第一行输入测试组数 T。每组输入一个字符串 s。

## 输出格式

每组输出 YES 或 NO。

## 自测样例

### 输入

```text
4
YES
esY
YesYes
se
```

### 输出

```text
NO
YES
YES
NO
```
