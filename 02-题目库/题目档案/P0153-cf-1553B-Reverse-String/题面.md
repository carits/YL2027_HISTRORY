# P0153 Reverse String 题面

## 题目来源

- 平台：CodeForces
- 题号：1553B
- 原题链接：https://codeforces.com/problemset/problem/1553/B
- VJudge 写法：`[problem:CodeForces-1553B]`

## 问题陈述

给定字符串 s 和目标串 t。判断能否从 s 的某个位置开始，先向右连续读若干字符，然后最多转向一次向左连续读若干字符，正好得到 t。

## 输入格式

第一行输入测试组数 q。每组输入 s 和 t。

## 输出格式

每组输出 YES 或 NO。

## 约束要点

枚举起点和转向位置即可。

## 样例输入

```text
4
abcdef
cde
abcdef
edc
abacaba
aaa
abc
ac
```

## 样例输出

```text
YES
YES
NO
NO
```
