# P0199 Multiset 题面

## 题目来源

- 平台：CodeForces
- 题号：1354D
- 原题链接：https://codeforces.com/problemset/problem/1354/D
- VJudge 写法：`[problem:CodeForces-1354D]`

## 问题描述

维护一个多重集合。正数查询表示插入该数；负数查询 `-k` 表示删除当前集合中的第 $k$ 小元素。所有操作结束后，输出集合中任意一个数；若为空输出 $0$。

## 输入格式

第一行输入 `n,q`。第二行输入初始有序多重集合的 $n$ 个数。第三行输入 $q$ 个操作。

## 输出格式

输出最终集合中的任意一个元素，若集合为空输出 $0$。

## 数据范围

`1 <= n,q <= 10^6`，元素值范围在 `1..n`。

## 样例输入

```text
5 5
1 2 3 4 5
-1 -1 -1 -1 -1
```

## 样例输出

```text
0
```
