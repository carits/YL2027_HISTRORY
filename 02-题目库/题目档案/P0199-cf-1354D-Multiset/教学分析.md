# P0199 Multiset 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 数据结构 |
| 子标签 | 树状数组, 第k小, 动态 multiset |
| 三维难度 | 3/4/4 |
| 综合等级 | L5 |
| 推荐阶段 | 提高- |
| 适合用途 | 树状数组应用题 |

## 训练价值

适合用来讲树状数组不只可以求和，还可以做动态频率表和 order statistics。

## 讲评重点

1. 先让学生复述题意和限制条件。
2. 明确核心状态、枚举对象或判定函数。
3. 单独讲边界和常见错误。
4. 最后再给代码，避免学生只背实现。
