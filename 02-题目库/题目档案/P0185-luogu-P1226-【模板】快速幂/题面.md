# P0185 【模板】快速幂 题面

## 问题陈述

给定三个整数 `a, b, p`，求 `a^b mod p`，并按题目要求输出表达式形式。

## 输入格式

输入一行三个整数 `a, b, p`。

## 输出格式

输出 `a^b mod p=s`，其中 `s` 是取模结果。

## 数据范围

`0 <= a,b < 2^31`，`a+b>0`，`2 <= p < 2^31`。

## 样例输入

```text
2 10 9
```

## 样例输出

```text
2^10 mod 9=7
```
