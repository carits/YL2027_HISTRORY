# P0185 【模板】快速幂 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 数学 |
| 子标签 | 快速幂, 取模运算, 二进制拆分 |
| 三维难度 | 2/2/1 |
| 综合等级 | L2 |
| 推荐阶段 | 普及- |
| 适合用途 | 模板题, 必讲题 |

## 训练价值

这是普及阶段必须熟练的模板，建议要求学生能手写并解释二进制拆分。

## 课堂讲解建议

1. 先让学生复述题意中的限制条件。
2. 再引导学生抽象出核心状态或判定函数。
3. 对照样例手推一遍，确认边界情况。
4. 最后再讲代码结构和复杂度。

## 二刷建议

完成本题后，可以要求学生不看题解重新写一遍，并单独解释为什么算法不会漏情况。
