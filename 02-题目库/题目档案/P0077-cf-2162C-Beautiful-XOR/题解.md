# P0077 Beautiful XOR 题解

## 一、题意简化

本题可以简化为：

```text
给定 a 和 b。一次操作可选择 0 到当前 a 之间的整数 x，并令 a=a xor x。要求在不超过 100 次操作内把 a 变成 b，或判断不可能。
```

## 二、算法设计

操作无法产生比当前最高位更高的位，因此若 b 的最高位更高则无解。若最高位相同，直接取 x=$a \oplus b$ 即可。若 b 最高位更低，先把 a 变成当前最高位以下全为 1 的数，再一步变成 b。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `构造` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

每组 O(1)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
