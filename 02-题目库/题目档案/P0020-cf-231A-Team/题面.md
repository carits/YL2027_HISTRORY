## 问题陈述

有 n 道题。每道题有三个人分别给出是否会做，1 表示会，0 表示不会。若至少两个人会做，则团队会解决这道题。求能解决多少道题。

## 输入格式

输入 n，接下来 n 行每行 3 个 0/1 整数。

## 输出格式

输出能解决的题目数量。

## 自测样例

### 输入

```text
3
1 1 0
1 1 1
1 0 0
```

### 输出

```text
2
```
