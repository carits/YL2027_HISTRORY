# P0180 Getting Points 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 二分答案 |
| 子标签 | 最少学习天数, 每周任务, 最大休息天数 |
| 三维难度 | 2/3/2 |
| 综合等级 | L3 |
| 推荐阶段 | 普及 |
| 适合用途 | 二分答案训练题 |

## 训练价值

适合讲“最大休息天数”反向转化为“最少学习天数”。

## 课堂讲解建议

1. 先让学生复述题意中的限制条件。
2. 再引导学生抽象出核心状态或判定函数。
3. 对照样例手推一遍，确认边界情况。
4. 最后再讲代码结构和复杂度。

## 二刷建议

完成本题后，可以要求学生不看题解重新写一遍，并单独解释为什么算法不会漏情况。
