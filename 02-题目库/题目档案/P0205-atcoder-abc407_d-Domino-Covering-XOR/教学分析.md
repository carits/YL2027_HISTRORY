# P0205 Domino Covering XOR 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 搜索 |
| 子标签 | DFS, 状态压缩, 多米诺覆盖 |
| 综合等级 | L3 |
| 三维难度 | 2/3/3 |

## 训练价值

本题适合训练学生把文字条件转化为明确的状态、判定或构造过程。讲评时应重点追问：为什么这样做不会漏情况，复杂度为什么能通过。

## 课堂讲解重点

1. 先复述题意，明确输入输出。
2. 找出关键不变量或单调性。
3. 手推样例，解释每一步状态变化。
4. 分析边界和反例。
5. 最后再给代码实现。

## 二刷建议

二刷时要求学生独立写出核心判断函数或核心数据结构，不只背代码。
