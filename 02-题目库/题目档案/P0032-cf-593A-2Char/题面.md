## 问题陈述

给定 n 个只含小写字母的字符串。你可以选择两个不同的小写字母，只保留那些所有字符都属于这两个字母的字符串。

求保留下来的字符串总长度最大是多少。

## 输入格式

第一行输入 n。

接下来 n 行，每行一个字符串。

## 输出格式

输出最大总长度。

## 数据范围

字符串只含小写英文字母。

## 样例输入

```text
4
aba
c
aa
bc
```

## 样例输出

```text
5
```
