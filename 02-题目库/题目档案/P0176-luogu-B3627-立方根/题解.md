# P0176 立方根 题解

## 一、题意简化

给定正整数 n，求 n 的整数立方根下取整，即最大的整数 x 满足 x^3≤n。

## 二、算法设计

答案具有单调性：若 x^3≤n，则更小的数也满足。对 x 进行整数二分，用 __int128 防止乘法溢出。

## 三、复杂度分析

O(log n)。

## 四、常见错误

- 多组数据没有清空数组、计数器或集合。
- 输出大小写、空格、换行不符合题目要求。
- 没有按数据范围选择合适的数据类型。
- 忽略极端数据，例如空结果、只有一个元素、全部相同、边界上限等。
