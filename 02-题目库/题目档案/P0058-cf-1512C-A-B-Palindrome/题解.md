# P0058 A-B Palindrome 题解

## 一、题意简化

给定 a、b 和一个由 0、1、? 构成的字符串。需要把所有 ? 替换为 0 或 1，使最终字符串是回文串，并且 0 的数量正好为 a，1 的数量正好为 b。

## 二、算法设计

先根据回文对称关系传播已知字符：如果一边是 0/1，另一边是 ?，就把 ? 填成相同字符；若两边已知且不同，则无解。

然后统计已有 0 和 1 的数量。对于两边都是 ? 的位置成对填入 00 或 11。

最后处理中间位置，并检查数量与回文性质。

## 三、复杂度分析

每组 O(n)。

## 四、易错点

- 没有严格按照多组数据格式读取。
- 下标从 0 开始和从 1 开始混用。
- 只过了样例，没有检查边界情况。
- 输出格式多空格或少换行。
