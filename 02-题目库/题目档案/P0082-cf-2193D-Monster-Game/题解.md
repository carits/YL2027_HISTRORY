# P0082 Monster Game 题解

## 一、题意简化

有 n 把剑，每把剑强度为 a_i；第 i 关怪物需要 b_i 次攻击。选择难度 x 后，只有强度至少 x 的剑可用，每把剑只能攻击一次。得分为 x×通关数，求最大得分。

## 二、算法设计

固定难度 x 时，可用剑数量为 cnt(x)。能通过的关数是最大的 k，使得 b 的前 k 项和不超过 cnt(x)。只需枚举 x 为某个剑的强度。将强度排序后维护可用数量，用前缀和二分通关数。

## 三、复杂度分析

O(n log n)。

## 四、常见错误

- 多组数据没有清空数组、计数器或集合。
- 输出大小写、空格、换行不符合题目要求。
- 没有按数据范围选择合适的数据类型。
- 忽略极端数据，例如空结果、只有一个元素、全部相同、边界上限等。
