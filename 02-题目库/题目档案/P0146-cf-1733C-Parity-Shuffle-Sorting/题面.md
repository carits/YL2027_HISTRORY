# P0146 Parity Shuffle Sorting 题面

## 题目来源

- 平台：CodeForces
- 题号：1733C
- 原题链接：https://codeforces.com/problemset/problem/1733/C
- VJudge 写法：`[problem:CodeForces-1733C]`

## 问题描述

给定数组。一次操作选择 `l < r`：若 `a_l + a_r` 为奇数，则令 `a_r = a_l`；否则令 `a_l = a_r`。要求用不超过 `n` 次操作把数组变成非降序，输出任意方案。

## 输入格式

第一行输入测试组数 `t`。每组输入 `n` 和数组 `a`。

## 输出格式

每组先输出操作次数，再输出每次操作的两个下标。

## 数据范围

`1 <= n <= 100000`，所有测试的 `n` 总和不超过 `100000`。

## 样例输入

```text
3
2
7 8
5
1 1000000000 3 0 5
1
0
```

## 样例输出

```text
1
1 2
4
1 5
1 2
3 5
1 4
0
```
