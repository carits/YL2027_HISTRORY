# P0182 Traffic Light 题面

## 问题陈述

有一个循环变化的交通灯字符串，当前位置颜色为给定字符 $c$。从每个颜色为 $c$ 的位置出发，向后等待直到遇到绿色 $g$。求最坏情况下需要等待的最长距离。若 $c$ 已经是绿色，答案为 0。

## 输入格式

第一行输入测试组数。每组输入 `n, c`，再输入长度为 $n$ 的字符串。

## 输出格式

每组输出一个整数。

## 数据范围

`1 <= n`，字符串只包含小写颜色字符，至少存在一个 $g$。

## 样例输入

```text
3
5 r
rggry
1 g
g
6 y
gygygy
```

## 样例输出

```text
3
0
1
```
