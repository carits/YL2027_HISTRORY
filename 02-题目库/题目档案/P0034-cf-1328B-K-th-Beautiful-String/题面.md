## 问题陈述

长度为 n 的字符串只由 a 和 b 构成，并且恰好有两个 b。把所有这样的字符串按字典序排列，求第 k 个字符串。

## 输入格式

第一行输入测试组数 T。

每组输入 n 和 k。

## 输出格式

每组输出一个长度为 n 的字符串。

## 数据范围

1 <= k <= n(n-1)/2。

## 样例输入

```text
3
5 1
5 2
3 3
```

## 样例输出

```text
aaabb
aabab
bba
```
