# P0034 K-th Beautiful String 题解

## 一、题意简化

长度为 n 的字符串只由 a 和 b 构成，并且恰好有两个 b。把所有这样的字符串按字典序排列，求第 k 个字符串。

## 二、算法设计

从字典序看，靠右的两个 b 更靠前。

设从右往左数，第一个 b 与结尾形成一组：

- 最右两位是 b 时，对应 1 个字符串；
- 第一个 b 再往左一位时，对应 2 个字符串；
- 再往左时，对应 3 个字符串。

不断用 k 减去当前组大小，找到第一个 b 所在组，再确定第二个 b 的位置。

## 三、复杂度分析

每组 O(n) 构造字符串。

## 四、易错点

- 没有严格按照多组数据格式读取。
- 下标从 0 开始和从 1 开始混用。
- 只过了样例，没有检查边界情况。
- 输出格式多空格或少换行。
