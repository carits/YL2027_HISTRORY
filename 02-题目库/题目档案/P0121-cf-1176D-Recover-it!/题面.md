# P0121 Recover it! 题面

## 题目来源

- 平台：CodeForces
- 题号：1176D
- 原题链接：https://codeforces.com/problemset/problem/1176/D
- VJudge 写法：`[problem:CodeForces-1176D]`

## 问题描述

某个数组 `a` 有 `n` 个数。根据每个 `a_i` 额外加入一个数形成数组 `b`：如果 `a_i` 是质数，就加入第 `a_i` 个质数；否则加入 `a_i` 的最大真因子。现在给出打乱后的 `b`，要求恢复任意一个可能的 `a`。

## 输入格式

第一行输入 `n`。第二行输入 `2n` 个整数，表示数组 `b`。

## 输出格式

输出 `n` 个整数，表示一种合法的原数组 `a`。

## 数据范围

`1 <= n <= 2 * 10^5`，`2 <= b_i <= 2750131`，保证有解。

## 样例输入

```text
3
3 5 2 3 2 4
```

## 样例输出

```text
3 4 2
```
