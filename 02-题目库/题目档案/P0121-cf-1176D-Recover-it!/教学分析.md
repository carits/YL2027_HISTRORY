# P0121 Recover it! 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 数论 |
| 子标签 | 质数表, 质数编号, 贪心恢复 |
| 三维难度 | 4/4/4 |
| 综合等级 | L5 |
| 推荐阶段 | 提高- |
| 适合用途 | 数论构造题, 贪心恢复题 |

## 训练价值

这题适合讲“生成规则反向恢复”，关键是从大到小处理，让大数先和它的小伙伴配对。

## 讲评重点

1. 先让学生复述题意和限制条件。
2. 明确核心状态、枚举对象或判定函数。
3. 单独讲边界和常见错误。
4. 最后再给代码，避免学生只背实现。
