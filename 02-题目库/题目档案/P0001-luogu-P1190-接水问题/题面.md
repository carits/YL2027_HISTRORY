# P0001 接水问题 题面

## 问题陈述

有 n 个人按给定顺序排队接水，有 m 个接水龙头。每个人接水所需时间为 t_i。
开始时，前 min(n,m) 个人分别占用空闲龙头。之后每当某个龙头上的人接完水，队伍中的下一个人立刻去这个龙头接水。

求所有人都接完水所需的最短总时间。

## 输入格式

第一行输入两个整数 n, m。
第二行输入 $n$ 个整数 t_1, t_2, ..., t_n，表示每个人接水所需时间。

## 输出格式

输出一个整数，表示全部接完水的时间。

## 自测样例

### 输入

```text
5 3
4 4 1 2 1
```

### 输出

```text
4
```
