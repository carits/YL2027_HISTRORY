# P0073 Shizuku Hoshikawa and Farm Legs 题解

## 一、题意简化

本题可以简化为：

```text
农场中只有鸡和牛，鸡有 2 条腿，牛有 4 条腿。给定总腿数 n，求可能的鸡牛数量配置数。
```

## 二、算法设计

若 n 为奇数无解。否则设牛有 c 头，c 可从 0 到 floor(n/4)，鸡的数量随之唯一确定，因此答案为 floor(n/4)+1。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `数学` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

每组 O(1)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
