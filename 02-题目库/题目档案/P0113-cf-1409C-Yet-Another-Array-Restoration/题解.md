# P0113 Yet Another Array Restoration 题解

## 一、题意简化

本题需要围绕 **构造**，处理 `等差数列, 枚举公差` 相关问题。

## 二、算法设计

因为 $x$ 和 $y$ 要同在等差数列中，所以公差 d 必须整除 y - x。

枚举所有可行公差 d，先放入从 x 到 y 的这一段，再尽量向左扩展正数项，最后向右补齐 n 项。

在所有方案中选择最大值最小的一组。

## 三、正确性说明

算法的核心是抓住题目中的不变量或最优选择依据，并在每一步只保留对后续有意义的信息。对于需要枚举的题目，枚举对象被限制在可证明的有限范围内；对于贪心题，选择依据保证不会让后续状态变差；对于动态规划题，状态转移完整覆盖所有可行路径。

## 四、复杂度分析

设 $D$ = y - x，枚举 D 的因子，整体复杂度约 O(因子数 * n)。

## 五、常见错误

- 没有单独处理边界输入。
- 使用 `int` 导致中间结果溢出。
- 枚举范围过大或过小。
- 多组数据时没有清空临时变量。
