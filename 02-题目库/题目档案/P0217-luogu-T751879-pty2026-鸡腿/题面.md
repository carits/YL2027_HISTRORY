## 鸡腿（leg）

#### 题目描述

在大海的深处，居住着一只英俊的胖头鱼，他非常向往胖头鸡的生活，所以向坤神许下了心愿，以两年半不准打篮球的代价，换取了一双胖头鸡的腿。

他打算去往胖头鸡的部落拜访朋友，以胖头鱼出发的位置作为原点 $(0, 0)$ 建立平面直角坐标系，则胖头鸡的部落在 $(n, m)$ 。假设某时刻胖头鱼所在的位置为 $(x, y)$ ，为了避免迷路，他每次只会**向右**移动**一个**单位到达 $(x + 1, y)$ 或者**向上**移动**一个**单位到达 $(x, y + 1)$ 。可惜胖头鱼还不能很熟练的使用鸡腿，所以他在不同位置进行移动时需要消耗的体力是不同的。具体来说，当 $x + y$ 是**偶数**时，向右移动一个单位需要消耗 $a$ 的体力，向上移动一个单位需要消耗 $b$ 的体力；当 $x + y$ 是**奇数**时，向右移动一个单位需要消耗 $c$ 的体力，向上移动一个单位需要消耗 $d$ 的体力。

胖头鱼想知道，为了到达胖头鸡的部落，他**最少**需要消耗多少体力。由于他实在没有力气思考了，所以只好求助于你。

#### 输入格式（文件名：leg.in）

只有一行，包含 $6$ 个整数，依次为 $n, m, a, b, c, d$ ，表示坐标以及在不同位置进行移动时需要消耗的体力。

#### 输出格式（文件名：leg.out）

输出一行，只有一个整数，表示最少需要消耗的体力。

#### 样例数据

存放在 `sample/leg` 目录下。

【样例 $1$ 输入】

存放在 `leg1.in` 文件中。

```
11 0 1 2 3 4
```

【样例 $1$ 输出】

存放在 `leg1.out` 文件中。

```
21
```

【样例 $1$ 解释】

只能不断的向右移动，其中 $6$ 次在 $x + y$ 为偶数的位置， $5$ 次在 $x + y$ 为奇数的位置，消耗的体力为 $6 \times 1 + 5 \times 3 = 21$ 。


【样例 $2$ 输入】

存放在 `leg2.in` 文件中。

```
5 3 2 1 5 3 
```

【样例 $2$ 输出】

存放在 `leg2.out` 文件中。

```
22
```

【样例 $2$ 解释】

最优的移动方式为 $(0, 0) \rightarrow (1, 0) \rightarrow (1, 1) \rightarrow (2, 1) \rightarrow (3, 1) \rightarrow (4, 1) \rightarrow (4, 2) \rightarrow (5, 2) \rightarrow (5, 3)$ 。

消耗的体力为 $2 + 3 + 2 + 5 + 2 + 3 + 2 + 3 = 22$ 。

【样例 $3, 4, 5$ 输入与输出】

存放在 `leg3.in` 与 `leg3.out` 、 `leg4.in` 与 `leg4.out` 、 `leg5.in` 与 `leg5.out` 文件中。

#### 数据说明

对于所有测试点，满足 $0 \le n, m, a, b, c, d \le 10^9$ 。

测试点的特殊属性如下：
  - 对于前 $20\%$ 的测试点，满足 $a = c$ 且 $b = d$ 。
  - 对于额外 $20\%$ 的测试点，满足 $a = b$ 且 $c = d$ 。
  - 对于额外 $20\%$ 的测试点，满足 $a < b$ 且 $c > d$ 。样例 $3$ 符合当前性质。
  - 对于额外 $20\%$ 的测试点，满足 $a < b$ 、 $c < d$ 且 $n > m$ 。样例 $4$ 符合当前性质。
  - 对于额外 $20\%$ 的测试点，没有特殊属性。样例 $5$ 符合当前性质。
