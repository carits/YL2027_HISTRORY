# P0187 逆序对 题面

## 问题陈述

给定一个长度为 $n$ 的正整数序列，求满足 `i < j` 且 `a_i > a_j` 的二元组数量，也就是逆序对数量。

## 输入格式

第一行输入 $n$。第二行输入 $n$ 个整数。

## 输出格式

输出逆序对个数。

## 数据范围

`1 <= n <= 500000`，元素值不超过 `10^9`。

## 样例输入

```text
5
3 1 2 5 4
```

## 样例输出

```text
3
```
