# P0187 逆序对 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 分治/树状数组 |
| 子标签 | 离散化, 树状数组, 逆序统计 |
| 三维难度 | 3/2/2 |
| 综合等级 | L3 |
| 推荐阶段 | 普及+ |
| 适合用途 | 树状数组训练题, 分治训练题 |

## 训练价值

可以同时作为离散化和树状数组的综合题。讲解时建议先用小样例手推从右往左的过程。

## 课堂讲解建议

1. 先让学生复述题意中的限制条件。
2. 再引导学生抽象出核心状态或判定函数。
3. 对照样例手推一遍，确认边界情况。
4. 最后再讲代码结构和复杂度。

## 二刷建议

完成本题后，可以要求学生不看题解重新写一遍，并单独解释为什么算法不会漏情况。
