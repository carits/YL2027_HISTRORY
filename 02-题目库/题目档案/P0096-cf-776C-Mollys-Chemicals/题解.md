# P0096 Molly's Chemicals 题解

## 一、题意简化

本题的核心是：给定长度为 n 的整数数组和整数 k。

## 二、算法设计

先枚举所有可能的 k 的幂，范围控制在前缀和可能达到的绝对值以内。

从左到右维护前缀和 sum。若当前前缀和为 sum，要找之前的前缀和 old，使得 sum - old = power，即 old = sum - power。

用哈希表统计每种前缀和出现次数。

## 三、正确性说明

算法直接围绕题目要求维护必要状态。每次转移或统计都只使用已经确定的信息，并且覆盖所有可能答案；最终取到的结果与题目定义一致。

## 四、复杂度分析

幂的数量很少，整体复杂度约为 O(n * 幂数量)。
