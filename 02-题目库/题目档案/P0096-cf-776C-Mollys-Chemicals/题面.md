## 问题陈述

给定长度为 n 的整数数组和整数 k。

统计有多少个连续子数组的和等于 k 的某个非负整数次幂，即 1, k, k^2, ... 中的某个值。

## 输入格式

第一行输入 n 和 k。

第二行输入 n 个整数 a_i。

## 输出格式

输出满足条件的子数组个数。

## 数据范围

n 可达 100000，数组元素可以为负。

## 样例输入

```text
5 2
1 1 1 1 1
```

## 样例输出

```text
11
```
