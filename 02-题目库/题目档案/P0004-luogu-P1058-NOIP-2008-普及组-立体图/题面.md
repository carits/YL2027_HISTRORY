# P0004 [NOIP 2008 普及组] 立体图

## 题目描述

给定一个 $m 	imes n$ 的格子区域，每个格子上堆放若干个边长为 $1$ 的正方体积木。请按照题目规定的透视方式，输出这些积木组成的立体图。

单个积木的显示模板如下，背景位置使用 `.`，积木内部空白保留为空格：

```text
..+---+
./   /|
+---+ |
|   | +
|   |/.
+---+..
```

若积木左右相邻，显示效果类似：

```text
..+---+---+
./   /   /|
+---+---+ |
|   |   | +
|   |   |/.
+---+---+..
```

若积木上下堆叠或前后相邻，需要按照相同的立体投影规则覆盖到字符矩阵中。位于第 $m$ 行第 $1$ 列格子中最底层积木的左下角顶点，是整张图最左下角的点。

## 输入格式

第一行包含两个整数 $m,n$，表示格子区域大小。

接下来 $m$ 行，每行包含 $n$ 个整数，第 $i$ 行第 $j$ 列的整数表示该格子上堆放的积木数量。

## 输出格式

输出题目要求的最小字符矩阵。矩阵中的背景位置用 `.` 表示，积木内部空白用空格表示。

## 数据范围

$1 \le m,n \le 50$，每个格子上的积木数在 $1$ 到 $100$ 之间。

## 样例

### 样例 1

**输入**

```text
3 4
2 2 1 2
2 2 1 1
3 2 1 2
```

**输出**

```text
......+---+---+...+---+
..+---+  /   /|../   /|
./   /|-+---+ |.+---+ |
+---+ |/   /| +-|   | +
|   | +---+ |/+---+ |/|
|   |/   /| +/   /|-+ |
+---+---+ |/+---+ |/| +
|   |   | +-|   | + |/.
|   |   |/  |   |/| +..
+---+---+---+---+ |/...
|   |   |   |   | +....
|   |   |   |   |/.....
+---+---+---+---+......
```
