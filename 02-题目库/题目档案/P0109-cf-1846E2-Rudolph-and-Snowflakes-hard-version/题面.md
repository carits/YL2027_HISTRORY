# P0109 Rudolph and Snowflakes (hard version) 题面

## 题目来源

- 平台：CodeForces
- 题号：1846E2
- 原题链接：https://codeforces.com/problemset/problem/1846/E2
- VJudge 写法：`[problem:CodeForces-1846E2]`

## 问题陈述

给定整数 n，判断它能否写成 1 + k + k^2 + ... + k^m 的形式。

其中 k > 1，m >= 2，也就是说至少包含 1、k、k^2 三项。

## 输入格式

第一行输入整数 T，表示测试组数。

接下来 T 行，每行一个整数 n。

## 输出格式

对每组数据，如果存在这样的 k 和 m，输出 YES，否则输出 NO。

## 数据范围

1 <= T，1 <= n <= 10^18。

## 样例输入

```text
5
7
13
31
10
21
```

## 样例输出

```text
YES
YES
YES
NO
YES
```
