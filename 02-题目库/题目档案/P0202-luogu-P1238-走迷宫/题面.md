# P0202 走迷宫 题面

## 来源

- 平台：洛谷

## 问题陈述

给定一个 m 行 n 列迷宫，1 表示可走，0 表示不可走。给出起点和终点，输出所有不重复经过点的路径；移动方向优先级为左、上、右、下。若没有路径，输出 -1。

## 输入格式

输入 m,n，随后输入迷宫矩阵，再输入起点和终点坐标。

## 输出格式

按要求输出所有路径，每个点形如 (x,y)，相邻点之间用 -> 连接；无路输出 -1。

## 样例输入

```text
3 3
1 1 1
0 1 0
1 1 1
1 1 3 3
```

## 样例输出

```text
(1,1)->(1,2)->(2,2)->(3,2)->(3,3)
```
