# P0202 走迷宫 题解

## 一、题意简化

本题可以简化为：

```text
给定一个 m 行 n 列迷宫，1 表示可走，0 表示不可走。给出起点和终点，输出所有不重复经过点的路径；移动方向优先级为左、上、右、下。若没有路径，输出 -1。
```

## 二、算法设计

用 DFS 搜索所有简单路径。递归时按左、上、右、下的顺序扩展，使用 visited 防止重复经过同一点。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `搜索` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

最坏可能指数级，符合本题输出所有路径的要求。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
