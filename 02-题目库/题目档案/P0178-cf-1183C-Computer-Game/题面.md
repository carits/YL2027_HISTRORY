# P0178 Computer Game 题面

## 问题陈述

有 $q$ 组询问。每局游戏有 $n$ 回合，初始电量为 $k$。每回合可以选择普通游玩，消耗 $a$ 电量；也可以边充边玩，消耗 $b$ 电量，其中 `b < a`。每一回合结束后电量必须仍然大于 0。求最多能有多少回合选择普通游玩；如果无论如何都无法完成 $n$ 回合，输出 `-1`。

## 输入格式

第一行输入 $q$。每组询问输入四个整数 `k, n, a, b`。

## 输出格式

每组询问输出一行答案。

## 数据范围

`1 <= q <= 100000`，`1 <= k,n <= 10^9`，`1 <= b < a <= 10^9`。

## 样例输入

```text
3
15 5 3 2
15 5 4 3
20 5 7 3
```

## 样例输出

```text
4
-1
1
```
