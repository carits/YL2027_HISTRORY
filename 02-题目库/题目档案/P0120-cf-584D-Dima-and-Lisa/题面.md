# P0120 Dima and Lisa 题面

## 题目来源

- 平台：CodeForces
- 题号：584D
- 原题链接：https://codeforces.com/problemset/problem/584/D
- VJudge 写法：`[problem:CodeForces-584D]`

## 问题陈述

给定整数 n，要求把 n 表示成尽量少的质数之和。

输出质数个数以及这些质数。

## 输入格式

输入一个整数 n。

## 输出格式

第一行输出质数个数。

第二行输出这些质数。

## 数据范围

n 为正整数。

## 样例输入

```text
27
```

## 样例输出

```text
3
3 5 19
```
