# P0120 Dima and Lisa 题解

## 一、题意简化

本题需要围绕 **数学**，处理 `质数, 哥德巴赫构造, Miller-Rabin` 相关问题。

## 二、算法设计

如果 n 本身是质数，答案是 1。

如果 n 是偶数，可以表示成两个质数之和，枚举其中一个质数即可。

如果 n 是奇数且 n - 2 是质数，则答案是 2：2 和 n - 2。

否则先取出 3，剩下的偶数再拆成两个质数。

## 三、正确性说明

算法的核心是抓住题目中的不变量或最优选择依据，并在每一步只保留对后续有意义的信息。对于需要枚举的题目，枚举对象被限制在可证明的有限范围内；对于贪心题，选择依据保证不会让后续状态变差；对于动态规划题，状态转移完整覆盖所有可行路径。

## 四、复杂度分析

使用快速素性测试，枚举拆分点，实际复杂度较低。

## 五、常见错误

- 没有单独处理边界输入。
- 使用 `int` 导致中间结果溢出。
- 枚举范围过大或过小。
- 多组数据时没有清空临时变量。
