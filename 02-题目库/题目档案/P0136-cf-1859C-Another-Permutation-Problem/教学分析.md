# P0136 Another Permutation Problem 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 贪心 |
| 子标签 | 排列构造, 枚举断点, 贡献计算 |
| 三维难度 | 2/3/2 |
| 综合等级 | L3 |
| 推荐阶段 | 普及+ |
| 适合用途 | 构造观察题 |

## 训练价值

适合讲“构造题也可以先枚举结构候选”，不要求学生一眼推出闭式公式。

## 讲评重点

1. 先让学生复述题意和限制条件。
2. 明确核心状态、枚举对象或判定函数。
3. 单独讲边界和常见错误。
4. 最后再给代码，避免学生只背实现。
