# P0184 Perfect Team 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 数学/贪心 |
| 子标签 | 组队, 取最小值, 资源约束 |
| 三维难度 | 1/2/1 |
| 综合等级 | L2 |
| 推荐阶段 | 入门-普及 |
| 适合用途 | 简单贪心公式题 |

## 训练价值

适合作为“多个必要上界取最小”的入门例题。

## 课堂讲解建议

1. 先让学生复述题意中的限制条件。
2. 再引导学生抽象出核心状态或判定函数。
3. 对照样例手推一遍，确认边界情况。
4. 最后再讲代码结构和复杂度。

## 二刷建议

完成本题后，可以要求学生不看题解重新写一遍，并单独解释为什么算法不会漏情况。
