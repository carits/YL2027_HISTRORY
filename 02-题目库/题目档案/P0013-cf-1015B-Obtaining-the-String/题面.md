# P0013 Obtaining the String 题面

## 来源

- 平台：CodeForces
- 题号：1015B
- 原题链接：https://codeforces.com/problemset/problem/1015/B

## 问题陈述

给出两个长度相同的字符串 s 和 t。一次操作可以交换 s 中相邻的两个字符。求能否把 s 变成 t；若可以，输出一种操作序列。

## 输入格式

输入 n、字符串 s、字符串 t。

## 输出格式

若无法变换，输出 -1；否则先输出操作次数，再输出每次交换的位置 p，表示交换 p 和 p+1 两个字符。

## 自测样例

### 输入

```text
4
abcd
cdab
```

### 输出

```text
4
2 1 3 2
```
