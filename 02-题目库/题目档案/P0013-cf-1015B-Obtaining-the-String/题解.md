# P0013 Obtaining the String 题解

## 一、题意简化

给出两个长度相同的字符串 s 和 t。一次操作可以交换 s 中相邻的两个字符。求能否把 s 变成 t；若可以，输出一种操作序列。

## 二、算法设计

如果两个字符串字符 multiset 不同，则无法变换。否则从左到右处理 t[i]，在 s 的 i 之后找到第一个等于 t[i] 的位置 j，然后不断相邻交换把它挪到 i。

## 三、复杂度分析

时间复杂度满足该题官方限制；详见题解中的循环/枚举范围分析。

## 四、易错点

- 没有按多组数据读取。
- 边界条件没有单独检查。
- 输出格式大小写不一致。
