# P0200 Strongly Composite 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 数论 |
| 子标签 | 质因数分解, 配对, 贪心分组 |
| 三维难度 | 3/3/2 |
| 综合等级 | L3 |
| 推荐阶段 | 普及+ |
| 适合用途 | 质因数分解训练题 |

## 训练价值

适合讲“质因子是资源”的贪心分组。可以用样例 `3,10,14` 手推。

## 课堂讲解建议

1. 先让学生复述题意中的限制条件。
2. 再引导学生抽象出核心状态或判定函数。
3. 对照样例手推一遍，确认边界情况。
4. 最后再讲代码结构和复杂度。

## 二刷建议

完成本题后，可以要求学生不看题解重新写一遍，并单独解释为什么算法不会漏情况。
