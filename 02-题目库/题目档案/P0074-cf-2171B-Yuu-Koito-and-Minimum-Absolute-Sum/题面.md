# P0074 Yuu Koito and Minimum Absolute Sum 题面

## 来源

- 平台：CodeForces
- 题号：2171B
- 原题链接：https://codeforces.com/problemset/problem/2171/B
- VJudge 写法：`[problem:CodeForces-2171B]`

## 问题陈述

给定一个含有 -1 的数组，-1 表示空位，可填非负整数。差分数组 b_i=a_{i+1}-a_i，要求最小化 |sum b_i|，并输出达到最小值中字典序最小的完整数组。

## 输入格式

第一行输入测试组数 T。每组输入 n 和数组 a。

## 输出格式

每组先输出最小值，再输出构造出的数组。

## 样例输入

```text
6
4
2 -1 7 1
4
-1 2 4 -1
8
2 -1 1 5 11 12 1 -1
3
-1 -1 -1
3
2 5 4
2
-1 5
```

## 样例输出

```text
1
2 0 7 1
0
0 2 4 0
0
2 0 1 5 11 12 1 2
0
0 0 0
2
2 5 4
0
5 5
```
