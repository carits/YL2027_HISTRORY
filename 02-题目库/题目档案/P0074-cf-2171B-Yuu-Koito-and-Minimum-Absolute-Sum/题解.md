# P0074 Yuu Koito and Minimum Absolute Sum 题解

## 一、题意简化

本题可以简化为：

```text
给定一个含有 -1 的数组，-1 表示空位，可填非负整数。差分数组 b_i=a_{i+1}-a_i，要求最小化 |sum b_i|，并输出达到最小值中字典序最小的完整数组。
```

## 二、算法设计

差分和会完全抵消成 a_n-a_1，因此目标只与首尾有关。内部空位为了字典序最小都填 0。若某个端点为空，就把它填成另一个端点；若两端都空，则都填 0。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `构造` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

每组 O(n)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
