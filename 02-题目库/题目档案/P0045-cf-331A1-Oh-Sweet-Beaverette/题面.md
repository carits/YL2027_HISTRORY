# P0045 Oh Sweet Beaverette 题面

## 题目描述

给定一列树的美观值，需要砍掉若干棵，使剩余树不少于两棵，且剩余序列第一棵和最后一棵美观值相同，同时剩余美观值总和最大。输出最大总和和一种砍树方案。

## 输入格式

输入 n 和长度为 n 的数组 a。A1 子任务中 n≤100。

## 输出格式

第一行输出最大总和和砍掉的树数；第二行输出砍掉的树的编号。

## 样例

### 输入

```text
5
1 2 3 1 2
```

### 输出

```text
8 1
1
```
