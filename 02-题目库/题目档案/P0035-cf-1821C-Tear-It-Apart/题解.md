# P0035 Tear It Apart 题解

## 一、题意简化

给定字符串 s。一次操作可以使某些位置逐步变成目标字符。对每个目标字符，最困难的是它之间隔开的最长非目标字符连续段。

## 二、算法设计

枚举最终变成的字符 c。

把 s 中不是 c 的连续段长度取最大值 g。每次操作相当于让这段中能被 c 影响的范围扩大，因此长度 g 需要不断减半。

计算 g 需要被除以 2 多少次才能变成 0，这就是选择 c 时的操作数。对 26 个字符取最小值。

## 三、复杂度分析

每组 O(26n)。

## 四、易错点

- 没有严格按照多组数据格式读取。
- 下标从 0 开始和从 1 开始混用。
- 只过了样例，没有检查边界情况。
- 输出格式多空格或少换行。
