## 问题陈述

给定字符串 s。一次操作可以使某些位置逐步变成目标字符。对每个目标字符，最困难的是它之间隔开的最长非目标字符连续段。

求最少操作次数，使整个字符串变成同一种字符。

## 输入格式

第一行输入测试组数 T。

每组输入一个字符串 s。

## 输出格式

每组输出最少操作次数。

## 数据范围

字符串只含小写英文字母。

## 样例输入

```text
3
aba
aaaa
abc
```

## 样例输出

```text
1
0
1
```
