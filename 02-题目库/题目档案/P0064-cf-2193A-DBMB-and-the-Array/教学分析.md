# P0064 DBMB and the Array 教学分析

## 一、教学价值

| 项目 | 内容 |
|---|---|
| 主标签 | 数学 |
| 子标签 | 取模, 构造可达性 |
| 三维难度 | 1/1/1 |
| 综合等级 | L1 |

## 二、课堂讲解重点

1. 先让学生复述题意，确认输入、输出和目标。
2. 引导学生发现核心性质：数组和每次只会增加 x，因此最终和必须不小于当前和，并且差值必须是 x 的倍数。
3. 强调为什么当前复杂度可以通过。
4. 用样例手推一遍，再写代码。

## 三、适合训练的能力

- 题意转化能力
- 边界条件检查
- 根据数据范围选择算法
- 代码实现稳定性

## 四、建议追问

- 这题最容易错在哪个边界？
- 暴力做法的复杂度是多少？
- 正解中最关键的一步为什么成立？
- 能否构造一个反例卡掉错误贪心或错误模拟？
