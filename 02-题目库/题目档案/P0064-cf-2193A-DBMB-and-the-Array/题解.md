# P0064 DBMB and the Array 题解

## 一、题意简化

本题可以简化为：

```text
给定一个长度为 n 的数组 a、目标和 s，以及操作步长 x。一次操作可以选择任意一个元素并给它加上 x。判断能否经过若干次操作后，使整个数组的元素和恰好等于 s。
```

## 二、算法设计

数组和每次只会增加 x，因此最终和必须不小于当前和，并且差值必须是 x 的倍数。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `数学` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

每组 O(n)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
