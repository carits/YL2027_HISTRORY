# P0163 Burning Midnight Oil 题面

## 题目来源

- 平台：CodeForces
- 题号：165B
- 原题链接：https://codeforces.com/problemset/problem/165/B
- VJudge 写法：`[problem:CodeForces-165B]`

## 问题陈述

输入 n 和 k。若选择初始工作量 v，则实际完成量为 v + floor(v/k) + floor(v/k^2) + ...。求最小的 v，使实际完成量至少为 n。

## 输入格式

输入 n 和 k。

## 输出格式

输出最小 v。

## 约束要点

函数随 v 单调不减。

## 样例输入

```text
7 2
```

## 样例输出

```text
4
```
