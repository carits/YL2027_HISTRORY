# P0063 Almost All Divisors 题解

## 一、题意简化

给定一个整数 x 的所有非平凡约数，也就是除了 1 和 x 本身以外的所有正约数，顺序被打乱。

## 二、算法设计

如果列表真的是 x 的全部非平凡约数，那么排序后最小约数和最大约数的乘积应为 x。

先令候选值 cand = d_min * d_max，并检查排序数组中对称位置乘积是否都等于 cand。

最后枚举 cand 的所有约数，去掉 1 和 cand，得到真正的非平凡约数列表，与输入排序后完全比较。

## 三、复杂度分析

排序 O(n log n)，枚举约数 O(sqrt(cand))。

## 四、易错点

- 没有严格按照多组数据格式读取。
- 下标从 0 开始和从 1 开始混用。
- 只过了样例，没有检查边界情况。
- 输出格式多空格或少换行。
