## 问题陈述

给定一个整数 x 的所有非平凡约数，也就是除了 1 和 x 本身以外的所有正约数，顺序被打乱。

要求还原 x。若不存在这样的 x，输出 -1。

## 输入格式

第一行输入测试组数 T。

每组输入 n，再输入 n 个整数，表示给出的约数列表。

## 输出格式

每组输出 x 或 -1。

## 数据范围

x 可能较大，需要使用 64 位整数。

## 样例输入

```text
3
2
2 3
3
2 3 4
4
2 4 8 16
```

## 样例输出

```text
6
-1
32
```
