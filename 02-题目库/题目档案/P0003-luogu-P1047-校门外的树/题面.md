# P0003 校门外的树 题面

## 来源

- 平台：洛谷

## 问题陈述

一条马路上有从 $0$ 到 $L$ 编号的 $L+1$ 棵树。给出若干个$[l,r]$，每个区间内的树都会被移走。求最后还剩多少棵树。

## 输入格式

第一行输入两个整数 L, M。
接下来 $M$ 行每行两个整数 l, r，表示要移走树的闭区间。

## 输出格式

输出一个整数，表示剩余树的数量。

## 自测样例

### 输入

```text
500 3
150 300
100 200
470 471
```

### 输出

```text
298
```
