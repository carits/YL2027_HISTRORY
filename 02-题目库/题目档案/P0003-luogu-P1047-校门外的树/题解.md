# P0003 校门外的树 题解

## 一、题意简化

本题核心可以归纳为：

```text
围绕 `模拟` 专题，完成 `闭区间处理, 标记数组`，并注意 `区间端点遗漏, 数组越界`。
```

## 二、算法设计

### 核心思路

- 主标签：模拟
- 子标签：区间标记, 数组模拟
- 关键技巧：闭区间处理, 标记数组

处理流程：

1. 读入题目数据。
2. 根据题意确定需要维护的状态或枚举对象。
3. 按照 `闭区间处理, 标记数组` 完成模拟、枚举或统计。
4. 输出最终答案。

## 三、复杂度分析

由于该题定位为 入门 阶段训练题，通常正解复杂度应满足原题限制。正式补全题面后，可在此处写明准确复杂度。

## 四、常见错误

- 区间端点遗漏, 数组越界
- 输入输出格式不严格。
- 边界条件没有单独检查。

## 五、教学讲法
