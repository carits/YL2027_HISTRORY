# P0003 校门外的树 题解

## 一、题意简化

一条马路上有从 0 到 L 编号的 L+1 棵树。给出若干个闭区间 [l,r]，每个区间内的树都会被移走。求最后还剩多少棵树。

## 二、算法设计

因为树的位置是离散的 0...L，可以直接打标记。更规范的写法是用差分：对每个区间 [l,r] 做 d[l]++, d[r+1]--，最后前缀和为 0 的位置就是未被移走的树。

## 三、复杂度分析

时间复杂度 O(L+M)，空间复杂度 O(L)。

## 四、易错点

- 忘记树的编号包含 0 和 L，一共有 L+1 棵。
- 区间是闭区间，r 位置也要移走。
