# P0171 砍树 题面

## 题目来源

- 平台：洛谷
- 题号：P1873
- 原题链接：https://www.luogu.com.cn/problem/P1873
- VJudge 写法：`[problem:洛谷-P1873]`

## 问题陈述

有 n 棵树，设置锯片高度 H 后，高于 H 的部分会被锯下。要求得到至少 m 长度的木材，并使 H 尽量高。求最大的整数 H。

## 输入格式

第一行输入 n 和 m。第二行输入 n 棵树的高度。

## 输出格式

输出最大 H。

## 约束要点

n 可达 10^6，需要 O(n log V) 处理。

## 样例输入

```text
4 7
20 15 10 17
```

## 样例输出

```text
15
```
