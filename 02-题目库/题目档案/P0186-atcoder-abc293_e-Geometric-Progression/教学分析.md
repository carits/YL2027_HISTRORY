# P0186 Geometric Progression 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 数学 |
| 子标签 | 矩阵快速幂, 等比数列求和, 模运算 |
| 三维难度 | 3/3/3 |
| 综合等级 | L4 |
| 推荐阶段 | 提高- |
| 适合用途 | 快速幂拓展题 |

## 训练价值

适合在快速幂后讲“矩阵把递推打包”。这题比裸快速幂更能体现建模价值。

## 课堂讲解建议

1. 先让学生复述题意中的限制条件。
2. 再引导学生抽象出核心状态或判定函数。
3. 对照样例手推一遍，确认边界情况。
4. 最后再讲代码结构和复杂度。

## 二刷建议

完成本题后，可以要求学生不看题解重新写一遍，并单独解释为什么算法不会漏情况。
