# P0186 Geometric Progression 题面

## 问题陈述

给定整数 `A, X, M`，求 `1 + A + A^2 + ... + A^(X-1)` 对 `M` 取模的结果。

## 输入格式

输入一行三个整数 `A, X, M`。

## 输出格式

输出一个整数，即所求和模 `M` 的值。

## 数据范围

`1 <= A,M <= 10^9`，`1 <= X <= 10^12`。

## 样例输入

```text
3 4 7
```

## 样例输出

```text
5
```
