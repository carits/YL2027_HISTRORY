## 问题描述

给定一个整数 $N$ 的十进制数字。把所有数字分成两组，并在每组内重新排列，形成两个不含前导零的正整数。求两个整数乘积的最大值。

## 输入格式

输入一个整数 $N$。

## 输出格式

输出最大乘积。

## 数据范围

`1 <= N <= 10^9`，且 $N$ 至少包含两个非零数字。

## 样例输入

```text
123
```

## 样例输出

```text
63
```
