# P0203 Select Mul 题解

## 一、题意简化

给定一个整数 `N` 的十进制数字。把所有数字分成两组，并在每组内重新排列，形成两个不含前导零的正整数。求两个整数乘积的最大值。

## 二、算法设计

枚举每个数字属于第一个数还是第二个数的子集。对每一组数字降序排列，得到该组能形成的最大数。两组都非空且不是全零时，更新乘积最大值。

## 三、正确性说明

对于固定的一组数字，要让形成的数最大，显然应按数字从大到小排列。枚举所有划分覆盖所有可能分组，因此取最大乘积即可。

## 四、复杂度分析

设位数为 `d`，时间复杂度 `O(2^d * d log d)`，`d <= 10`，空间复杂度 `O(d)`。
