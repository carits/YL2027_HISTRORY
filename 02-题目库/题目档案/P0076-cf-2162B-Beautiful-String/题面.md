# P0076 Beautiful String 题面

## 来源

- 平台：CodeForces
- 题号：2162B
- 原题链接：https://codeforces.com/problemset/problem/2162/B
- VJudge 写法：`[problem:CodeForces-2162B]`

## 问题陈述

给定长度不超过 10 的二进制串 s。需要选择一个子序列 p，使 p 非降；删除 p 后剩余字符串为回文。输出任意一种方案，若不存在输出 -1。

## 输入格式

第一行输入测试组数 T。每组输入 n 和字符串 s。

## 输出格式

若存在方案，输出选择位置个数 $k$ 和 $k$ 个下标；否则输出 -1。

## 样例输入

```text
2
3
010
3
001
```

## 样例输出

```text
0
2
1 2
```
