# P0076 Beautiful String 题解

## 一、题意简化

本题可以简化为：

```text
给定长度不超过 10 的二进制串 s。需要选择一个子序列 p，使 p 非降；删除 p 后剩余字符串为回文。输出任意一种方案，若不存在输出 -1。
```

## 二、算法设计

n 很小，直接枚举所有子集作为被删除的子序列，检查选中字符是否非降、剩余字符是否回文。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `暴力` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

每组 O(2^n n)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
