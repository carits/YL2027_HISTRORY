# P0154 Watering an Array 题面

## 题目来源

- 平台：CodeForces
- 题号：1917C
- 原题链接：https://codeforces.com/problemset/problem/1917/C
- VJudge 写法：`[problem:CodeForces-1917C]`

## 问题描述

有数组 `a`。接下来 `d` 天每天必须二选一：要么按周期序列 `v` 给前 `b_i` 个元素加一，要么统计满足 `a_j=j` 的位置数量加入得分并把整个数组清零。求最大得分。

## 输入格式

第一行输入测试组数。每组输入 `n,k,d`，数组 `a`，以及长度为 `k` 的周期数组 `v`。

## 输出格式

每组输出最大得分。

## 数据范围

`1 <= n <= 2000`，`1 <= k <= 100000`，`k <= d <= 10^9`，所有测试的 `n` 总和不超过 `2000`。

## 样例输入

```text
2
3 4 4
1 2 3
1 3 2 3
5 1 1
0 5 0 5 0
5
```

## 样例输出

```text
4
0
```
