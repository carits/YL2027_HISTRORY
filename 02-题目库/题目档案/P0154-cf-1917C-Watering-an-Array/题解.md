# P0154 Watering an Array 题解

## 一、题意简化

有数组 $a$。接下来 $d$ 天每天必须二选一：要么按周期序列 $v$ 给前 $b_i$ 个元素加一，要么统计满足 `a_j=j` 的位置数量加入得分并把整个数组清零。求最大得分。

## 二、算法设计

枚举第一次执行统计操作前做了多少次加一操作。只需要枚举前 `min(d,2n)` 次，因为再往后单次统计的潜在收益不会优于尽早统计。若当前有 `cnt` 个位置满足 `a_i=i`，统计后剩余 `d-i-1` 天最多每两天再获得 1 分，所以候选为 `cnt + (d-i-1)/2`。

## 三、正确性说明

每次清零后，要想再次得分，至少需要一天执行加一，再用一天统计，因此剩余部分的上界是每两天 1 分。枚举第一次统计时机并模拟前若干次加法，可以覆盖所有可能更优的开局。超过 `2n` 次继续等待不会比提前统计更优。

## 四、复杂度分析

时间复杂度 $O(n * min(d,2n))$，空间复杂度 $O(n)$。
