# P0086 Specialty String 题解

## 一、题意简化

给定字符串，每次可以删除当前相邻且相等的一对字符。判断是否存在操作顺序使整个字符串全部删除。

## 二、算法设计

把删除相邻相等字符看作括号式消除。顺序扫描字符串，用栈维护当前未消除字符：若栈顶等于当前字符就弹出，否则压入。最终栈空则可以全部消除。

## 三、复杂度分析

O(n)。

## 四、常见错误

- 多组数据没有清空数组、计数器或集合。
- 输出大小写、空格、换行不符合题目要求。
- 没有按数据范围选择合适的数据类型。
- 忽略极端数据，例如空结果、只有一个元素、全部相同、边界上限等。
