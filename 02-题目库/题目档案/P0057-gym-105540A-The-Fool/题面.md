# P0057 The Fool

## 题目描述

有一个 $n 	imes m$ 的字符网格，每个格子由一个长度为 $k$ 的字符串表示。除一个格子外，其余所有格子的字符串完全相同；Neeko 所在格子的字符串至少有一个字符与其他格子不同。

请找出 Neeko 所在的格子位置。

## 输入格式

第一行包含三个整数 $n,m,k$。

接下来 $n$ 行，每行包含 $m\cdot k$ 个可见 ASCII 字符。第 $i$ 行中第 $(j-1)k+1$ 到第 $jk$ 个字符组成第 $(i,j)$ 个格子的字符串。

保证没有额外空格或换行，并且答案唯一。

## 输出格式

输出两个整数 $r,c$，表示 Neeko 位于第 $r$ 行第 $c$ 列。

## 数据范围

$2 \le n,m \le 200$，$1 \le k \le 10$。

## 样例

### 样例 1

**输入**

```text
3 5 3
QWQQWQQWQQWQQWQ
QWQQWQQWQQWQQWQ
QWQQWQQWQQWQQwQ
```

**输出**

```text
3 5
```
