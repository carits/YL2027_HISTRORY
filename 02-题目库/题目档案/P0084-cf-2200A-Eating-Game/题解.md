# P0084 Eating Game 题解

## 一、题意简化

本题可以简化为：

```text
n 个玩家围成一圈，第 i 个玩家有 a_i 份食物。选择任意玩家先开始，之后按环形顺序轮流行动；轮到一个仍有食物的玩家时，他吃掉一份。吃掉最后一份食物的玩家获胜。求有多少名玩家可能成为胜者。
```

## 二、算法设计

n 和 a_i 都很小，直接枚举每个起点并模拟整场过程，用集合记录赢家。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `模拟` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

每组 O(n sum a_i)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
