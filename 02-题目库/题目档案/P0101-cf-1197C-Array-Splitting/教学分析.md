# P0101 Array Splitting 教学分析

## 一、教学定位

| 项目 | 内容 |
|---|---|
| 推荐阶段 | 普及- |
| 综合等级 | L3 |
| 三维难度 | 2/3/1 |
| 主标签 | 贪心 |
| 子标签 | 排序数组, 最大间隔, 分组代价 |

## 二、训练价值

- 训练学生把题意转化为可维护的状态或统计对象。
- 适合用于 贪心 专题中的讲评、作业或二刷。
- 可以配合相邻题目比较不同写法的复杂度差异。

## 三、课堂讲法

1. 先让学生复述题意中的“要求输出什么”。
2. 明确暴力做法的复杂度瓶颈。
3. 引导学生找出可以预处理、贪心、统计或转化的部分。
4. 最后强调代码边界和数据类型。

## 四、二刷建议

二刷时要求学生不看题解重新写出核心公式或核心状态，并补充至少两个边界样例。
