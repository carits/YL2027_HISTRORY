# P0101 Array Splitting 题解

## 一、题意简化

本题的核心是：给定一个非降序数组 a，要求把它分成 k 个非空连续段。

## 二、算法设计

若不切分，总代价是 a_n - a_1。

在相邻两个数之间切一刀，就可以省去这个相邻差值。

要切 k-1 刀，所以选择最大的 k-1 个相邻差值省掉。

## 三、正确性说明

算法直接围绕题目要求维护必要状态。每次转移或统计都只使用已经确定的信息，并且覆盖所有可能答案；最终取到的结果与题目定义一致。

## 四、复杂度分析

排序相邻差值 O(n log n)。原数组已按题意给定为非降序。
