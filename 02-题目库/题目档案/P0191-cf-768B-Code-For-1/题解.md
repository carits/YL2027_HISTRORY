# P0191 Code For 1 题解

## 一、题意简化

定义一个数 `n` 对应的序列：若 `n <= 1`，序列就是 `[n]`；否则序列为 `seq(n/2) + [n mod 2] + seq(n/2)`。给定 `n,l,r`，求该序列第 `l` 到第 `r` 位中 1 的个数。

## 二、算法设计

递归统计。先计算 `seq(n)` 的长度。递归函数处理当前节点对应区间 `[L,R]`，若与查询区间无交集返回 0；若 `n<=1` 返回该位是否为 1；否则递归左右子序列，并检查中间位。

## 三、正确性说明

序列定义本身是一棵二叉递归树。每个节点的左右子树完全相同，中间位为 `n mod 2`。递归时只访问与查询区间相交的节点，所有相交叶子和中间位的贡献之和就是答案。

## 四、复杂度分析

递归深度 `O(log n)`，访问节点数量 `O(log n)` 到 `O(log n + 覆盖分支数)`，空间为递归栈。
