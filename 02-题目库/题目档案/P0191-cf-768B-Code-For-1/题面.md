# P0191 Code For 1 题面

## 问题陈述

定义一个数 `n` 对应的序列：若 `n <= 1`，序列就是 `[n]`；否则序列为 `seq(n/2) + [n mod 2] + seq(n/2)`。给定 `n,l,r`，求该序列第 `l` 到第 `r` 位中 1 的个数。

## 输入格式

输入 `n, l, r`。位置从 1 开始。

## 输出格式

输出指定区间中 1 的个数。

## 数据范围

`n` 和位置范围可达 64 位整数范围内的递归序列长度。

## 样例输入

```text
10 4 12
```

## 样例输出

```text
6
```
