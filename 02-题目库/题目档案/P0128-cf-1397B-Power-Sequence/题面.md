# P0128 Power Sequence 题面

## 题目来源

- 平台：CodeForces
- 题号：1397B
- 原题链接：https://codeforces.com/problemset/problem/1397/B
- VJudge 写法：`[problem:CodeForces-1397B]`

## 问题陈述

给定 $n$ 个正整数。选择一个整数 c >= 1，使排序后的数组尽量接近序列 1, c, c^2, ..., c^{n-1}。

代价为对应位置绝对差之和。求最小代价。

## 输入格式

第一行输入 n。

第二行输入 n 个正整数。

## 输出格式

输出最小代价。

## 数据范围

1 <= n，数组元素为正整数。

## 样例输入

```text
3
1 3 9
```

## 样例输出

```text
0
```
