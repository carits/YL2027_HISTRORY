# P0207 Peaceful Teams 题面

## 题目来源

- 平台：AtCoder
- 题号：abc310_d
- 原题链接：https://atcoder.jp/contests/abc310/tasks/abc310_d
- VJudge 写法：`[problem:AtCoder-abc310_d]`

## 问题描述

有 `N` 名选手，要分成 `T` 个非空队伍。给出若干对不相容选手，同一队不能包含任意一对不相容选手。求满足条件的分队方案数，队伍不区分编号。

## 输入格式

输入 `N,T,M`，之后 `M` 行输入不相容的二元组。

## 输出格式

输出方案数。

## 数据范围

`1 <= T <= N <= 10`，`0 <= M <= N(N-1)/2`。

## 样例输入

```text
5 2 2
1 3
3 4
```

## 样例输出

```text
4
```
