# P0194 Creative Snap 题面

## 问题陈述

基地长度为 `2^n`，有 `k` 个复仇者位于若干位置。摧毁一个区间若没有复仇者，代价为 `A`；若有 `cnt` 个复仇者，直接摧毁代价为 `B * cnt * 区间长度`。也可以把区间分成两半分别处理。求摧毁整个基地的最小代价。

## 输入格式

输入 `n, k, A, B`，随后输入 `k` 个复仇者位置。

## 输出格式

输出最小代价。

## 数据范围

`1 <= n <= 30`，`1 <= k <= 100000`。

## 样例输入

```text
2 2 1 2
1 3
```

## 样例输出

```text
6
```
