# P0194 Creative Snap 教学分析

## 教学定位

| 项目 | 内容 |
|---|---|
| 主标签 | 分治 |
| 子标签 | 区间递归, 二分, 代价最小化 |
| 三维难度 | 3/4/3 |
| 综合等级 | L4 |
| 推荐阶段 | 提高 |
| 适合用途 | 分治优化题 |

## 训练价值

适合作为分治与代价函数结合的提高题。重点讲“空区间直接返回 A”这个剪枝。

## 课堂讲解建议

1. 先让学生复述题意中的限制条件。
2. 再引导学生抽象出核心状态或判定函数。
3. 对照样例手推一遍，确认边界情况。
4. 最后再讲代码结构和复杂度。

## 二刷建议

完成本题后，可以要求学生不看题解重新写一遍，并单独解释为什么算法不会漏情况。
