## 问题陈述

给定 n 名同学的初始语文成绩。

接下来有若干次操作，每次给区间 [x, y] 内所有同学的成绩同时加上 z，z 可以为负。

所有操作结束后，输出最终成绩的最小值。

## 输入格式

第一行输入 n 和 p。

第二行输入 n 个初始成绩。

接下来 p 行，每行输入 x、y、z，表示区间加 z。

## 输出格式

输出所有操作后的最低成绩。

## 数据范围

1 <= n, p。

## 样例输入

```text
5 3
1 2 3 4 5
1 3 2
2 5 -1
4 4 3
```

## 样例输出

```text
3
```

样例用于说明输入输出格式与核心计算过程，讲评时可结合题解手推。
