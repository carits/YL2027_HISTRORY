# P0041 汉诺塔问题 题面

## 题目描述

输入盘子数量 n 和三根杆子的字符编号，按照汉诺塔最少步数方案输出每一次移动。盘子从小到大编号为 1 到 n。

## 输入格式

输入一个整数 n，后面跟三个字符，分别表示起始杆、辅助杆、目标杆。

## 输出格式

每次移动一行，格式为 `编号:起点->终点`。

## 样例

### 输入

```text
3 a b c
```

### 输出

```text
1:a->c
2:a->b
1:c->b
3:a->c
1:b->a
2:b->c
1:a->c
```
