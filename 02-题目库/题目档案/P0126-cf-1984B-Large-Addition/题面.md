# P0126 Large Addition 题面

## 题目来源

- 平台：CodeForces
- 题号：1984B
- 原题链接：https://codeforces.com/problemset/problem/1984/B
- VJudge 写法：`[problem:CodeForces-1984B]`

## 问题描述

一个数字称为“大数字”，当且仅当它的每一位都在 $5$ 到 $9$ 之间。给定整数 $x$，判断它是否可能表示为两个位数相同的大正整数之和。

## 输入格式

第一行输入测试组数 $t$。每组输入一个整数 $x$。

## 输出格式

每组输出 `YES` 或 `NO`。

## 数据范围

`1 <= t <= 10000`，`10 <= x <= 10^18`。

## 样例输入

```text
5
1337
200
10
69
119
```

## 样例输出

```text
YES
NO
YES
NO
NO
```
