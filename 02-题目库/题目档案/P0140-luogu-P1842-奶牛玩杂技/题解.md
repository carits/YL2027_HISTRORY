# P0140 奶牛玩杂技 题解

## 一、题意简化

本题可以简化为：

```text
有 n 头牛，每头牛有体重 W_i 和力量 S_i。把所有牛从上到下摞成一列。某头牛的压扁指数等于它上方所有牛的总重量减去自己的力量。求一种排列，使最大压扁指数尽可能小。
```

## 二、算法设计

考虑相邻两头牛交换前后对最大风险的影响，可以推出应按 W_i+S_i 从小到大排序。排序后从上到下扫描，维护上方总重量并更新答案。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `贪心` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

O(n log n)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
