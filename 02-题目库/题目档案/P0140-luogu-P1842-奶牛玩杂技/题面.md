# P0140 奶牛玩杂技 题面

## 来源

- 平台：洛谷
- 题号：P1842
- 原题链接：https://www.luogu.com.cn/problem/P1842
- VJudge 写法：`[problem:洛谷-P1842]`

## 问题陈述

有 n 头牛，每头牛有体重 W_i 和力量 S_i。把所有牛从上到下摞成一列。某头牛的压扁指数等于它上方所有牛的总重量减去自己的力量。求一种排列，使最大压扁指数尽可能小。

## 输入格式

第一行输入 n。接下来 n 行每行输入 W_$i$ 和 $S$_i。

## 输出格式

输出最小可能的最大压扁指数。

## 样例输入

```text
3
10 3
2 5
3 3
```

## 样例输出

```text
2
```
