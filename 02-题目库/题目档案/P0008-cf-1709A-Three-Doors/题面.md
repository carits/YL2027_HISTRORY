# P0008 Three Doors 题面

## 来源

- 平台：CodeForces
- 题号：1709A
- 原题链接：https://codeforces.com/problemset/problem/1709/A
- VJudge 写法：`[problem:CodeForces-1709A]`

## 问题陈述

有 3 个盒子，编号 1 到 3。你一开始有一把编号为 x 的钥匙。打开编号为 x 的盒子后，里面可能有下一把钥匙；钥匙编号为 0 表示没有钥匙。判断能否连续打开全部 3 个盒子。

## 输入格式

每组测试数据输入一个整数 x，接着输入 3 个整数 a_1,a_2,a_3，a_i 表示第 i 个盒子里的钥匙编号。

## 输出格式

每组输出 YES 或 NO。

## 自测样例

### 输入

```text
3
1
2 3 0
1
0 3 2
2
3 1 0
```

### 输出

```text
YES
NO
YES
```
