# P0008 Three Doors 题解

## 一、题意简化

有 3 个盒子，编号 1 到 3。你一开始有一把编号为 x 的钥匙。打开编号为 x 的盒子后，里面可能有下一把钥匙；钥匙编号为 0 表示没有钥匙。判断能否连续打开全部 3 个盒子。

## 二、算法设计

从当前钥匙 x 开始模拟开盒子。每打开一个盒子，已开数量加一，然后把钥匙更新为盒子里的编号。若能打开 3 次，则成功。

## 三、复杂度分析

时间复杂度满足该题官方限制；详见题解中的循环/枚举范围分析。

## 四、易错点

- 没有按多组数据读取。
- 边界条件没有单独检查。
- 输出格式大小写不一致。
