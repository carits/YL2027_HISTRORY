# P0188 Box in Box 题解

## 一、题意简化

本题可以简化为：

```text
有 N 个长方体盒子，每个盒子有三条边。每个盒子都可以任意旋转。判断是否存在两个盒子，使其中一个盒子的三条边都严格小于另一个盒子的三条边。
```

## 二、算法设计

先把每个盒子的三条边从小到大排序。问题变成三维严格偏序。按第一维升序处理，同一第一维不能互相匹配；用树状数组维护已经处理过的较小第一维盒子中，第二维前缀范围内的最小第三维。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `数据结构` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

O(N log N)。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
