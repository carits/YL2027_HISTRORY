# P0210 ABC Puzzle 题面

## 来源

- 平台：AtCoder
- 题号：abc326_d
- 原题链接：https://atcoder.jp/contests/abc326/tasks/abc326_d
- VJudge 写法：`[problem:AtCoder-abc326_d]`

## 问题陈述

构造一个 N×N 网格，格子可以填 A、B、C 或点号。每一行和每一列都必须恰好包含 A、B、C 各一次。给定字符串 R 和 C，要求每行从左到右第一个非点字符分别为 R_i，每列从上到下第一个非点字符分别为 C_j。输出任意合法网格或判断无解。

## 输入格式

输入 N，随后输入字符串 R 和 C。

## 输出格式

若有解，输出 Yes 和网格；否则输出 No。

## 样例输入

```text
3
ABC
ABC
```

## 样例输出

```text
Yes
ABC
BCA
CAB
```
