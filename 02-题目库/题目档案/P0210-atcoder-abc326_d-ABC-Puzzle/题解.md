# P0210 ABC Puzzle 题解

## 一、题意简化

本题可以简化为：

```text
构造一个 N×N 网格，格子可以填 A、B、C 或点号。每一行和每一列都必须恰好包含 A、B、C 各一次。给定字符串 R 和 C，要求每行从左到右第一个非点字符分别为 R_i，每列从上到下第一个非点字符分别为 C_j。输出任意合法网格或判断无解。
```

## 二、算法设计

预处理所有满足当前行首个非点字符要求的行状态。逐行回溯，维护每列 A/B/C 是否已经出现以及每列第一个非点字符是否满足 C。

## 三、实现步骤

1. 读入数据。
2. 根据题意维护必要状态。
3. 按 `搜索` 的核心思路计算答案。
4. 严格按题目要求输出。

## 四、复杂度分析

N≤5，行状态数量很小，回溯可通过。

## 五、常见错误

- 没有单独处理边界数据。
- 多组数据时状态没有清空。
- 输出格式多空格、少换行或大小写错误。
- 使用了不满足数据范围的复杂度。
