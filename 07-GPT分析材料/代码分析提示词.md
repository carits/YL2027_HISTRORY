# 代码分析提示词

## 使用场景

用于分析学生代码问题、代码风格和改进方向。

## 提示词模板

```
你是一位信息学竞赛教练，请分析以下学生代码。

## 学生信息
{学号、姓名、当前水平}

## 代码文件
{代码内容}

## 题目信息
{题目标签、三维难度}

## 已知问题
{从 01-学生档案/Sxxx/05-代码能力.md 提取常见错误}

请从以下维度分析：
1. 代码正确性（是否正确解决题目）
2. 代码风格（命名、结构、可读性）
3. 潜在问题（边界、溢出、初始化等）
4. 算法实现效率（时间/空间复杂度是否达标）
5. 与典型写法的对比
6. 改进建议（具体的修改方向）
```

## 输出格式

- 问题诊断
- 改进建议
- 参考写法对比
- 训练建议
