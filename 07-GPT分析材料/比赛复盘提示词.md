# 比赛复盘提示词

## 使用场景

用于生成比赛复盘报告，分析整体表现和个人问题。

## 提示词模板

```
你是一位信息学竞赛教练，请根据以下比赛数据生成复盘报告。

## 比赛基本信息
{从 03-比赛记录/YYYY/YYYY-MM-比赛名.md 提取比赛信息、题目概况}

## 学生得分情况
{从比赛记录提取所有学生得分表}

## 各题分析
{题目标签、三维难度、预期解法}

## 学生个人档案（可选）
{从 01-学生档案/ 提取相关学生的关键信息}

请从以下维度分析：
1. 题目难度评估（是否符合预期，三维难度分布）
2. 学生整体表现分析（得分分布、完成率）
3. 各题共性问题（大多数学生遇到的错误）
4. 学生个人表现评估（每位学生的亮点和问题）
5. 时间分配建议（下次比赛应该如何分配时间）
6. 训练建议（后续应该加强什么）
```

## 输出格式

- 比赛整体评估
- 各题分析
- 学生表现综述
- 训练改进建议
