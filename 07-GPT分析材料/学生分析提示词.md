# 学生分析提示词

## 使用场景

用于生成学生阶段性分析报告，输入学生档案数据，输出分析报告。

## 提示词模板

```
你是一位信息学竞赛教练，请根据以下学生数据生成阶段性分析报告。

## 学生基本信息
{从 01-学生档案/Sxxx/00-学生总档案.md 提取}

## 知识点掌握情况
{从 01-学生档案/Sxxx/01-知识点掌握.md 提取}

## 近期做题记录
{从 01-学生档案/Sxxx/04-做题情况.md 提取}

## 比赛表现
{从 01-学生档案/Sxxx/02-比赛记录.md 提取}

## 代码能力
{从 01-学生档案/Sxxx/05-代码能力.md 提取}

## 学习习惯
{从 01-学生档案/Sxxx/06-学习习惯与性格.md 提取}

请从以下维度分析：
1. 知识点掌握情况评估（已掌握/基本掌握/学习中/未学习的分布）
2. 三维难度分析（在知识点/思维/实现哪个维度容易出错）
3. 常见错误模式
4. 学习习惯评估
5. 综合等级判断
6. 下阶段训练建议（具体到知识点和题目难度）
7. 需要老师重点关注的问题
```

## 输出格式

分析报告建议包含：
- 整体评估
- 分维度分析
- 具体训练建议
- 风险提示
