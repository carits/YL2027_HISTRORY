# 家长反馈提示词

## 使用场景

用于生成家长沟通内容，将学生表现转化为家长易懂的反馈。

## 提示词模板

```
你是一位信息学竞赛教练，请根据以下学生数据生成家长反馈内容。

## 学生基本信息
{从 01-学生档案/Sxxx/00-学生总档案.md 提取}

## 近期学习进展
{从 01-学生档案/Sxxx/08-阶段总结.md 提取}

## 知识点掌握情况
{从 01-学生档案/Sxxx/01-知识点掌握.md 提取}

## 比赛表现
{从 01-学生档案/Sxxx/02-比赛记录.md 提取}

## 学习习惯
{从 01-学生档案/Sxxx/06-学习习惯与性格.md 提取}

## 家长关注点
{从 01-学生档案/Sxxx/07-家长沟通.md 提取历史关注点}

请生成家长反馈，要求：
1. 语言通俗易懂，避免过多专业术语
2. 先说进步和亮点，再说需要改进的地方
3. 给出具体的、可操作的建议
4. 包含下阶段学习目标和计划
5. 如有需要家长配合的事项，明确说明
```

## 输出格式

- 学习进展概述
- 亮点与进步
- 需要改进的方面
- 下阶段目标
- 家长配合建议
