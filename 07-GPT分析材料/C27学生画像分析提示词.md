# C27 学生画像分析提示词

## 单个学生分析

请根据以下学生档案，从信息学竞赛培养角度进行分析：

1. 学生当前阶段。
2. 学生主要优势。
3. 学生主要短板。
4. 家庭配合情况。
5. 升学与承诺风险。
6. 是否建议重点培养。
7. 未来一个月训练建议。
8. 家长沟通建议。

要求：

- 不要夸大承诺。
- 不要只看分数，要结合习惯、备注和约谈。
- 风险信息只做教学管理建议，不做人格评价。

## 学生池分析

请根据 C27 小升初学生池汇总，输出：

1. S/A/B/C/R 分层名单。
2. 每个学生的推荐跟进动作。
3. 需要马上约谈的学生。
4. 需要补测评的学生。
5. 需要风险干预的学生。
6. 下一阶段班级训练安排建议。
