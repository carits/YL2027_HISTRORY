# 学生分层

## 分层标准

| 层级 | 代号 | 说明 |
|---|---|---|
| S层 | S | 核心种子，重点维护，给予承诺或保校 |
| A层 | A | 重点培养，天赋或习惯评分较高 |
| B层 | B | 重点观察，有潜力但需持续跟进 |
| C层 | C | 普通跟进，稳定训练即可 |
| R层 | R | 风险关注，有外流、诚信或训练管理风险 |
