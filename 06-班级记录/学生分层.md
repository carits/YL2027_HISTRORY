# 学生分层

## 分层标准

### C27小升初分层标准

| 层级 | 代号 | 说明 |
|---|---|---|
| S层 | S | 核心种子，重点维护，给予承诺或保校 |
| A层 | A | 重点培养，天赋或习惯评分较高 |
| B层 | B | 重点观察，有潜力但需持续跟进 |
| C层 | C | 普通跟进，稳定训练即可 |
| R层 | R | 风险关注，有外流或学习风险 |

### 常规OI分层标准

| 层级 | 综合等级 | 说明 |
|---|---|---|
| A层 | L6+ | 省选方向，可独立攻克难题 |
| B层 | L4-L5 | 提高组水平，需要系统专题训练 |
| C层 | L2-L3 | 普及组水平，需要巩固基础 |
| D层 | L1 | 入门阶段，需要基础语法训练 |
