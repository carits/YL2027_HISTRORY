# 比赛记录规范

## 一、目录结构

```
03-比赛记录/
├─ 比赛总表.md（索引，含比赛编号）
├─ 比赛记录规范.md
└─ 2026/
    ├─ C2026-001-春季模拟赛1/
    │   ├─ 00-比赛总览.md
    │   ├─ 01-题目列表.md
    │   ├─ 02-成绩总表.md
    │   ├─ 03-学生逐题表现.md
    │   ├─ 04-代码与提交索引.md
    │   ├─ 05-赛后复盘.md
    │   └─ 06-后续安排.md
    └─ C2026-002-胖头鱼杯/
        └─ （同上结构）
```

## 二、比赛编号规则

- 格式：`C{年份}-{序号}`
- 示例：C2026-001、C2026-002
- 目录命名：`C{年份}-{序号}-{比赛简称}`
- 简称规则：中文，无空格，如"春季模拟赛1"、"胖头鱼杯"

## 三、文件命名规范

比赛目录内的文件统一使用数字前缀命名，便于排序：

| 文件名 | 内容 |
|---|---|
| 00-比赛总览.md | 比赛基本信息、整体结果 |
| 01-题目列表.md | 题目详情、题目库编号关联 |
| 02-成绩总表.md | 按赛制的成绩表、分数分布 |
| 03-学生逐题表现.md | 每个学生的逐题得分、错因、代码链接 |
| 04-代码与提交索引.md | 代码存放路径、版本索引 |
| 05-赛后复盘.md | 整体复盘、各题分析、共性问题 |
| 06-后续安排.md | 讲评、补题、训练衔接、档案更新 |

## 四、比赛类型分类

| 类型 | 说明 |
|---|---|
| 模拟赛 | 日常训练模拟赛 |
| 周测 | 课堂周测 |
| 官方赛 | CSP、NOIP、NOI 等正式比赛 |
| 选拔赛 | 校内选拔 |
| 校外赛 | 外部比赛或招生观察 |

## 五、赛制分类

| 赛制 | 说明 | 成绩表格式 |
|---|---|---|
| OI | 传统OI赛制，按题得分 | 各题得分 + 总分 |
| ACM | ACM赛制，AC数+罚时 | AC数 + 罚时 |
| IOI | IOI赛制，部分分+交互 | 各题得分（含部分分） |

## 六、题目库关联

比赛题目必须关联题目库内部编号：

| 题号 | 题目库编号 | 示例 |
|---|---|---|
| A | P0030 | A题对应题目库 P0030 |
| B | P0102 | B题对应题目库 P0102 |

关联后可统计：
- 比赛中某知识点题目的整体表现
- 某学生在某类题目上的比赛表现
- 某题是否已在题目库中

## 七、代码档案关联

比赛代码存放路径：

```
05-代码档案/按学生/{学号}-{姓名}/比赛代码/C{年份}-{序号}/{题号}-{题目库编号}/
```

代码命名规范：
- 原始赛时代码：`original-v{版本号}-{状态}.cpp`
- 订正代码：`corrected-v{版本号}-{状态}.cpp`
- 二刷代码：`rebrush-v{版本号}-{状态}.cpp`

## 八、更新日志

比赛总表末尾维护更新日志：

| 日期 | 变更类型 | 变更内容 | 变更人 |
|---|---|---|---|
| YYYY-MM-DD | 新增/修改 | （具体描述） | （填写） |
