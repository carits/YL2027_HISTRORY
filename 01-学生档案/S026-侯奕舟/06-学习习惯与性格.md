# S026-侯奕舟 学生评价与培养建议补充

> 说明：本文件只写综合判断和培养建议，不放聊天证据，不罗列原始消息。

## 一、综合判断

1. 老师评价其特别愿意钻研题目，也坐得住，这是非常重要的学习品质。
2. 当前思维资质暂不如最强一批学生，但耐心和钻研能力可以支撑长期提升。
3. 学生适合走稳扎稳打路线，不宜用过高难题快速筛掉。
4. 培养上应鼓励深度思考和复盘，让其优势转化为成绩。
5. 如果后续训练强度保持稳定，可能是后劲型学生。

## 二、建议更新到原档案的位置

- `06-学习习惯与性格.md`：更新学习态度、课堂状态、专注稳定性、抗挫情况。
- `05-代码能力.md`：如上文涉及复杂代码、实现稳定性、调试习惯，应同步补充。
- `03-训练记录.md`：如上文涉及出勤、补课、作业、周训练，应同步补充。
- `08-阶段总结.md`：将本次结论作为阶段观察，不要写成最终定论。

## 三、培养建议

1. 对已有明确优势的学生：尽快用训练成绩和代码质量验证，不只停留在主观评价。
2. 对材料不足的学生：先补观察，不提前贴强弱标签。
3. 对存在外部去向、时间冲突或升学诉求强的学生：学习评价和家长沟通要同步推进。
4. 对有习惯问题的学生：优先抓笔记、作业、错题、复盘、出勤这些可执行指标。
5. 对高潜学生：每次比赛后补充“赛前状态—赛中表现—赛后复盘—下一步训练”。
