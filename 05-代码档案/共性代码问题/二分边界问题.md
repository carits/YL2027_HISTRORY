# 二分边界问题

## 问题描述

二分查找或二分答案时，边界条件写错，导致死循环或漏解。

## 典型错误代码

```cpp
// 死循环
while (l < r) {
    int mid = (l + r) / 2;
    if (check(mid)) l = mid;  // 应该用 l = mid + 1 或 mid = (l+r+1)/2
    else r = mid;
}
```

## 正确写法

```cpp
// 找第一个满足条件的位置
while (l < r) {
    int mid = (l + r) / 2;
    if (check(mid)) r = mid;
    else l = mid + 1;
}

// 找最后一个满足条件的位置
while (l < r) {
    int mid = (l + r + 1) / 2;
    if (check(mid)) l = mid;
    else r = mid - 1;
}
```

## 常见场景

- `l = mid` 时未用 `(l + r + 1) / 2` 导致死循环
- 循环条件写错（`<=` vs `<`）
- 实数二分精度不够
