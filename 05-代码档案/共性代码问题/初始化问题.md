# 初始化问题

## 问题描述

变量、数组或数据结构未正确初始化，导致使用垃圾值。

## 典型错误代码

```cpp
int ans; // 未初始化
int cnt[N]; // 全局变量自动初始化为0，但局部变量不会
```

## 正确写法

```cpp
int ans = 0;
int cnt[N] = {}; // C++11 列表初始化
// 或使用 memset
memset(cnt, 0, sizeof(cnt));
```

## 常见场景

- 局部变量未初始化
- 多组数据时未清空
- DP 数组未初始化
- 全局变量被上次数据污染
