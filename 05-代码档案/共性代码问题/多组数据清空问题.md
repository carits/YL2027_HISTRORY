# 多组数据清空问题

## 问题描述

多组测试数据时，上一组的数据未清空，影响当前组的结果。

## 典型错误代码

```cpp
int T;
cin >> T;
while (T--) {
    // 忘记清空数组、vector、map 等
    // 导致上一组数据残留
}
```

## 正确写法

```cpp
int T;
cin >> T;
while (T--) {
    // 清空所有需要重置的数据结构
    memset(vis, 0, sizeof(vis));
    vec.clear();
    mp.clear();
    // 然后处理当前组
}
```

## 常见场景

- 图的邻接表未清空
- 访问标记数组未重置
- 全局变量未恢复初始值
- vector/map 未 clear
