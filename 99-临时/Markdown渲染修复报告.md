## 已完成修复的类型

- **变量下标格式化**：`a_1` → `$a_1$`，`W_i` → `$W_i$`
- **复杂度表达式**：`O(n)` → `$O(n)$`，`O(n log n)` → `$O(n \log n)$`
- **嵌套复杂度**：`O((n+q)log n)` → `$O((n+q)\log n)$`
- **幂次表达式**：`10^6` → `$10^6$`，`2^n` → `$2^n$`
- **比较运算符**：`<=` → `$\le$`，`>=` → `$\ge$`
- **数学函数**：`min(n,m)` → `$\min(n,m)$`，`sqrt(n)` → `$\sqrt{n}$`
- **嵌套格式清理**：`$($...$)$` → `$...$`
- **双美元符号清理**：`$$O(1)$$` → `$O(1)$`
- **单独变量名**：`给定 a` → `给定 $a$`，`n 为奇数` → `$n$ 为奇数`
- **区间表示**：`闭区间 [l,r]` → `闭区间 $[l,r]$`
- **范围描述**：`从 0 到 L` → `从 $0$ 到 $L$`
- **xor运算**：`a xor b` → `$a \oplus b$`
- **gcd函数**：`gcd` → `$\gcd$`
- **反引号变量**：`` `n` `` → `$n$`
- **反引号复杂度**：`` `O(n log n)` `` → `$O(n \log n)$`
- **序数词变量**：`第 k 次` → `第 $k$ 次`
- **量词变量**：`n 个整数` → `$n$ 个整数`
