```text
题目档案.md
```

## 二、渲染规范

- 样例输入、样例输出、图形模板和代码均使用围栏代码块。
- C++ 代码统一使用 `cpp` 代码块。
- 数学表达式统一使用 GitHub Markdown 可识别的 `$...$` 或 `$$...$$` 格式。
- Markdown 表格列数保持一致。
- 未使用裸露的 `\(`、`\)`、`\[`、`\]` 数学定界符。
- 文件末尾均保留空行。

## 三、内容状态

所有纳入本包的题目均包含真实题面、题解思路和可编译 C++17 代码；没有使用 `TODO`、`待补充`、`#error` 作为占位内容。
