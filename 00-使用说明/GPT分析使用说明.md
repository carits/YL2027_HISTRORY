# GPT 分析使用说明

## 概述

`07-GPT分析材料/` 目录下存放各类 AI 辅助分析的提示词模板，配合本仓库的数据使用，可生成学生分析报告、比赛复盘、训练计划等。

## 提示词列表

| 文件 | 用途 | 输入数据来源 |
|---|---|---|
| 学生分析提示词.md | 生成学生阶段性分析报告 | 01-学生档案/ |
| 代码分析提示词.md | 分析学生代码问题和风格 | 05-代码档案/ |
| 比赛复盘提示词.md | 生成比赛复盘报告 | 03-比赛记录/ |
| 训练计划提示词.md | 生成个性化训练计划 | 01-学生档案/ + 02-题目库/ |
| 家长反馈提示词.md | 生成家长沟通内容 | 01-学生档案/ + 06-班级记录/ |

## 使用流程

1. 根据分析目的选择对应提示词
2. 从仓库中提取相关数据（学生档案、做题记录、比赛记录等）
3. 将提示词 + 数据发送给 GPT
4. 审核生成结果，根据实际情况调整
5. 将最终结果存入对应位置（如阶段总结存入学生档案目录）

## 数据提取建议

- 提取学生数据时，包含该学生的完整档案目录内容
- 提取题目数据时，包含三维难度和标签信息
- 提取比赛数据时，包含所有学生的得分和做题情况
- 数据量较大时，可只提取最近一个阶段的数据

## 注意事项

- GPT 生成的内容需要老师审核后才能使用
- 学生隐私数据不要发送到公开 API
- 提示词可根据实际使用效果持续优化
- 每次使用后记录提示词的调整，方便后续迭代
