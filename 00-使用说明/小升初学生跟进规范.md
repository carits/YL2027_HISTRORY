# 小升初学生跟进规范

## 一、学生分层

| 层级 | 名称 | 标准 |
|---|---|---|
| S | 核心种子 | 天赋高、习惯好、家长配合、升学价值高 |
| A | 重点培养 | 综合条件较好，值得重点投入 |
| B | 重点观察 | 有潜力，但信息不完整或存在不确定性 |
| C | 普通跟进 | 保持记录，暂不投入重点资源 |
| R | 风险关注 | 存在游戏、AI代写、外部去向、沟通失联等风险 |

## 二、跟进动作

| 状态 | 动作 |
|---|---|
| 核心种子 | 建立单独培养计划，每月复盘 |
| 重点培养 | 安排训练计划，跟踪比赛与作业 |
| 重点观察 | 补充测评、补充家长沟通、观察稳定性 |
| 普通跟进 | 保留基本信息，等待后续表现 |
| 风险关注 | 明确规则，与家长同步，观察是否改善 |

## 三、约谈记录要求

每次约谈必须记录：

1. 约谈时间。
2. 面谈人。
3. 谈话对象。
4. 沟通主题。
5. 家长诉求。
6. 教师判断。
7. 承诺事项。
8. 后续动作。

## 四、风险记录要求

风险记录必须客观，不写情绪化评价。

推荐写法：

```text
2026-xx-xx：发现训练期间存在游戏管理问题，已与家长沟通，后续观察训练投入情况。
```

不推荐写法：

```text
孩子不行，管不住，不能要。
```

## 五、更新频率

- 重点培养 / 承诺学生：每 2-4 周更新一次。
- 重点观察学生：每月更新一次。
- 普通跟进学生：有比赛、约谈、训练变化时更新。
- 风险关注学生：每次出现关键事件后立即更新。
