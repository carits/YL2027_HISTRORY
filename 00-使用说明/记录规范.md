# 记录规范

## 文件命名

| 类型 | 格式 | 示例 |
|---|---|---|
| 学生目录 | `S{编号}-{姓名}` | `S001-张三` |
| 比赛记录 | `{年份}-{月份}-{比赛名称}.md` | `2026-05-提高组模拟赛第1场.md` |
| 周训练 | `{年份}年第{周数}周训练.md` | `2026年第19周训练.md` |
| 代码文件 | `{平台}_{题号}.{扩展名}` | `luogu_P3372.cpp` |
| 阶段总结 | `{年份}-{月份}-阶段总结.md` | `2026-05-阶段总结.md` |

## 日期格式

统一使用 `YYYY-MM-DD`，如 `2026-05-10`

## 内容格式

- 每个 md 文件必须有标题（一级标题）
- 表格使用 markdown 标准格式
- 标签用逗号分隔，如 `线段树, 懒标记, 区间加`
- 列表项使用 `-` 符号
- 代码块标注语言类型，如 ` ```cpp `

## 编号规则

- 学生编号：S001、S002、S003... 顺序递增
- 题目编号：P0001、P0002... 顺序递增（系统内部编号）
- 比赛编号：C0001、C0002... 顺序递增
