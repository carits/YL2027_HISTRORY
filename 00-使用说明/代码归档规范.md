# 代码归档规范

## 代码存放位置

```
05-代码档案/
├─ 按学生/
│  ├─ S001-张三/
│  │  ├─ 比赛代码/     比赛中提交的代码
│  │  ├─ 训练代码/     日常训练的代码
│  │  ├─ 个人模板/     学生自己整理的代码模板
│  │  └─ 代码风格总结.md
│  └─ S002-李四/
└─ 共性代码问题/       常见代码错误汇总
```

## 代码文件命名

格式：`{平台}_{题号}.{扩展名}`

示例：
- `luogu_P3372.cpp`
- `cf_242E.cpp`
- `ac_abc123_d.cpp`

## 比赛代码命名

格式：`{比赛编号}_{题号}.{扩展名}`

示例：
- `C0001_T1.cpp`
- `C0001_T2.cpp`

## 代码文件头部

每个代码文件建议包含以下头部注释：

```cpp
// 题目：{题目名}
// 平台：{平台名}  题号：{题号}
// 标签：{知识点标签}
// 三维难度：知识{X} / 思维{X} / 实现{X}
// 日期：YYYY-MM-DD
// 备注：{如有}
```

## 共性代码问题

`05-代码档案/共性代码问题/` 下记录学生常见的代码错误模式，包括：

- 数组越界
- long long 问题
- 初始化问题
- 二分边界问题
- 多组数据清空问题
- 线段树 pushdown 问题
- 代码结构混乱问题

每个文件记录：问题描述、典型错误代码、正确写法、出现频率、涉及学生。

## 个人模板

学生自行整理的代码模板存放在 `05-代码档案/按学生/{学号}/个人模板/` 下，按专题分类，如：

- `线段树模板.cpp`
- `最短路模板.cpp`
- `数论模板.cpp`
